###### SortableList

- Проп `padding` переведен на числовые значения
    - `'3xs'` -> `2`
    - `'2xs'` -> `4`
    - `'xs'` -> `8`
    - `'s'` -> `12`
    - `'m'` -> `16`
    - `'l'` -> `20`
- Проп `borderRadius` переведен на числовые значения
    - `'m'` -> `8`
    - `'l'` -> `12`
    - `'xl'` -> `16`
