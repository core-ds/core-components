##### Types

- Тип `GapType` переведен на числовые значения:
    - `'3xs'` -> `2`
    - `'2xs'` -> `4`
    - `'xs'` -> `8`
    - `'s'` -> `12`
    - `'m'` -> `16`
    - `'l'` -> `20`
    - `'xl'` -> `24`
    - `'2xl'` -> `32`
    - `'3xl'` -> `40`
    - `'4xl'` -> `48`
    - `'5xl'` -> `64`
    - `'6xl'` -> `72`
    - `'7xl'` -> `96`
    - `'8xl'` -> `128`
