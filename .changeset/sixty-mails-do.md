###### Underlay

- Проп `padding` переведен на числовой тип. Для перехода используйте следующие соответствия:
    - `'3xs'` -> `2`
    - `'2xs'` -> `4`
    - `'xs'` -> `8`
    - `'s'` -> `12`
    - `'m'` -> `16`
    - `'l'` -> `20`
    - `'xl'` -> `24`
    - `'2xl'` -> `32`
    - `'3xl'` -> `40`
    - `'4xl'` -> `48`
    - `'5xl'` -> `64`
    - `'6xl'` -> `72`

- Проп `borderSize` на числовой тип. Для перехода используйте следующие соответствия:
    - `'xs'` -> `2`
    - `'s'` -> `4`
    - `'m'` -> `8`
    - `'l'` -> `12`
    - `'xl'` -> `16`
    - `'xxl'` -> `20`
    - `'3xl'` -> `24`
