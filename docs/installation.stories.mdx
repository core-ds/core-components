# Подключение

Каждый компонент публикуется отдельным пакетом, поэтому вы можете подключить только нужный, не устанавливая библиотеку целиком.

```bash
// установить всю библиотеку
yarn add @balafla/core-components

// установить отдельный компонент
yarn add @balafla/core-components-button
```

## Использование

```jsx
// импорт после установки отдельного пакета
import { Button } from '@balafla/core-components-button';

// импорт после установки всей библиотеки
import { Button } from '@balafla/core-components/button';
```
