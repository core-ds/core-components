## Иконки

Все иконки из Figma доступны для использования в [библиотеке иконок в виде React компонентов](https://github.com/core-ds/icons).

Для удобства воспользуйтесь [витриной иконок с поиском](https://core-ds.github.io/icons-demo/).

Иконки из пакета [icons-glyph](https://github.com/core-ds/icons) одноцветные и легко могут быть перекрашены с помощью `color`.
