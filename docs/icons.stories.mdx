# Иконки

Все иконки из Figma доступны для использования в [библиотеке иконок в виде React компонентов](https://github.com/core-ds/icons).

Для удобства воспользуйтесь [витриной иконок с поиском](https://core-ds.github.io/icons-demo/).

Иконки из пакета [icons-glyph](https://github.com/core-ds/icons) одноцветные и легко могут быть перекрашены с помощью `color`.

Есть два варианта использования иконок: Пакет [icons-glyph](https://github.com/core-ds/icons) и [CDNIcon](?path=/docs/CDNIcon--docs).

Если нет необходимости часто заменять одну иконку на другую, то подойдет [icons-glyph](https://github.com/core-ds/icons).
Вариант с [CDNIcon](?path=/docs/CDNIcon--docs) более гибкий.
Он дает возможность заменять иконку, имя которой можно получить, например, с backend, не вмешиваясь в код на фронте.
