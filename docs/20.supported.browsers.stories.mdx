# Поддерживаемые браузеры

Мы поддерживаем две последние стабильные версии всех популярных браузеров. Исключение — IE11+ и Android 5+ (Mobile Chrome).

## Desktop

-   Chrome
-   Yandex
-   Firefox
-   Edge
-   IE 11+
-   Safari

## Mobile

-   Android 5+
-   iOS
