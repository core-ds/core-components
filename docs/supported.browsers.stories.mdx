# Поддерживаемые браузеры

Мы поддерживаем две последние стабильные версии всех популярных браузеров. Исключение — Android 6+ (Mobile Chrome).

## Desktop

-   Chrome 113+
-   Yandex (целевой браузер)
-   Firefox
-   Edge
-   Safari

## Mobile

-   Android 6+
-   iOS 14+

## Важные замечания

-   **Минимальная версия Chrome**: 113
-   **Целевой браузер**: Яндекс Браузер (на базе Chromium)
-   **Chrome 79 и ниже**: Не поддерживаются из-за критических проблем совместимости
-   **Chrome 80-112**: Частичная поддержка с ограничениями

Подробная информация о проблемах совместимости и их решениях доступна в [гайде по траблшутингу для старых версий Chrome](./chrome-legacy-troubleshooting.md).
