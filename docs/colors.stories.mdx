### Использование цветов

При выделении любого элемента в Figma, в правом сайдбаре можно увидеть название цвета:

<img src='./images/colors_figma-element.png' width={319} alt='Пример цвета в Figma' />

Переменная var(--color-light-text-primary) соответствует цвету light/text/primary. Для получения соответствующих значений переменных токенов используйте [конвертер токенов](https://core-ds.github.io/core-components/master/?path=/docs/token-converter--docs).

```css
@import '@alfalab/core-components/vars/colors-bluetint.css';

.item {
    color: var(--color-light-text-primary);
}
```

> Если в Figma не видно название цвета или для него не оказалось переменной, спросите у дизайнера почему :)

### Модификации цветов

Для состояний интерактивных элементов (hover, active) могут использоваться модификации цветов: затемнение, осветление или изменение прозрачности.

Рекомендуем использовать [color-mod()](https://github.com/jonathantneal/postcss-color-mod-function) (в arui-scripts уже подключен).

Примеры использования:

-   затемнение → `color-mod(var(--color) shade(30%))` или `color-mod(var(--color) blenda(black 30%))`
-   осветление → `color-mod(var(--color) tint(30%))` или `color-mod(var(--color) blenda(white 30%))`
-   прозрачность → `color-mod(var(--color) alpha(30%))`

### Тени

Для теней предусмотрены отдельные переменные: [core-components/vars/shadows-bluetint.css](https://github.com/core-ds/core-components/blob/master/packages/vars/src/shadows-bluetint.css).

Названия видны в Figma аналогично цветам, смотрите размер (xs/s/m/l/xl):

-   XL → `var(--shadow-xl)`
-   XL Hard → `var(--shadow-xl-hard)`
