## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getStepsTestIds`.
Импорт из `@alfalab/core-components/steps/shared`.

Функция возвращает объект:

```jsx
{
    steps: dataTestId,
    step: `${dataTestId}-step`,
};
```
