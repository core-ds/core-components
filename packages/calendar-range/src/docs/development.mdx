
## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getCalendarRangeTestIds`.
Импорт из `@alfalab/core-components/calendar-range/shared`.

Функция возвращает объект:

```jsx
{
    component: dataTestId,
    calendarContainerFrom: `${dataTestId}-container-from`,
    calendarContainerTo: `${dataTestId}-container-to`,
};
```
