## Размеры

Кнопка доступна в размерах 40, 32, 24, 20.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='32' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='24' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='20' uppercase={false}>
            Всё в порядке
        </Status>
    </div>
</Container>
```

## Форма

Для статуса доступно два варианта скругления углов.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='40' uppercase={false} shape='rounded'>
            Всё в порядке
        </Status>
    </div>
</Container>
```

## Стиль лейбла

Для лейбла можно включить опцию `uppercase`.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='40'>Всё в порядке</Status>
    </div>
</Container>
```

## Аддон

В левый аддон можно передать иконку или другой контент.

```jsx live
render(() => {
    return (
        <Space direction='horizontal' size={12}>
            <Status view='muted' size={40} uppercase={false}>
                Без аддона
            </Status>
            <Status
                view='muted'
                size={40}
                uppercase={false}
                leftAddons={<DiamondsMIcon width={16} height={16} />}
            >
                С аддоном
            </Status>
        </Space>
    );
});
```

## Кастомизация

Статусу можно задать кастомный цвет текста и фона.

```jsx live
render(() => {
    return (
        <Container>
            <style>
                {
                    '.className.className {color: var(--color-light-decorative-text-purple); background-color: var(--color-light-decorative-muted-purple);}'
                }
            </style>
            <Status size={40} uppercase={false} className='className'>
                Кастом
            </Status>
        </Container>
    );
});
```
