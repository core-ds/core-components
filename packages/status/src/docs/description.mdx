## Размеры

Кнопка доступна в размерах 40, 32, 24, 20.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='32' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='24' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='20' uppercase={false}>
            Всё в порядке
        </Status>
    </div>
</Container>
```

## Форма

Для статуса доступно два варианта скругления углов.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='40' uppercase={false} shape='rounded'>
            Всё в порядке
        </Status>
    </div>
</Container>
```

## Стиль лейбла

Для лейбла можно включить опцию `uppercase`.

```jsx live
<Container>
    <div style={{ display: 'flex' }}>
        <Status size='40' uppercase={false}>
            Всё в порядке
        </Status>
        <Gap size='12' direction='horizontal' />
        <Status size='40'>Всё в порядке</Status>
    </div>
</Container>
```
