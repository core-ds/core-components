## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.  
Для удобного поиска элементов можно воспользоваться функцией `getActionButtonTestIds`.  
Импорт из `@alfalab/core-components/action-button/shared`.

Функция возвращает объект:

```jsx
{
    actionButton: dataTestId, 
    spinner: `${dataTestId}-loader`,
};
```
