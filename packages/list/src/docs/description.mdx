## Анатомия

Компонент состоит из кастомизируемого слота под маркер и текстовой части.

```jsx live
render(() => {
    const styleContent = {
        width: '24px',
        height: '24px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    };

    return (
        <Container>
            <Row align='middle'>
                <div style={styleContent}></div>
                <Typography.Text view='primary-medium'>Пример текстовой части</Typography.Text>
            </Row>
        </Container>
    );
});
```

## Примеры

        <List.Item caption='Пример подписи'>Пример элемента списка</List.Item>
