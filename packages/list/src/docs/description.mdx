## Использование в Webview

[Описание компонента для Webview](https://www.figma.com/file/2qkmYHFmV1rEmJyb6cEsW9/Web-%3A%3A-Mobile-Components?node-id=1153%3A300671)

При использовании списков в Webview можно использовать следующий сниппет.

Чтобы избежать дублирования и упростить ипользование компонента — можно создать на основе `List` преднастроенный компонент в своей локальной библиотеке.


```jsx live
// import { Space } from '@alfalab/core-components/space';
// import { Typography, typographyPresets } from '@alfalab/core-components/typography';
// import { listPres } from '@alfalab/core-components/list';

<List {...listPresets.mobile}>
    <Space size={2}>
        <Typography.Text {...typographyPresets.mobile.list.text.primary}>
            Накопительные счета
        </Typography.Text>
        <Typography.Text {...typographyPresets.mobile.list.text.secondary}>
            Индивидуальное обслуживание в любом отделении в России
        </Typography.Text>
    </Space>

    <Space size={2}>
        <Typography.Text  {...typographyPresets.mobile.list.text.primary}>
            Депозиты
        </Typography.Text>
        <Typography.Text {...typographyPresets.mobile.list.text.secondary}>
            Повышенные % ставки по депозитам
        </Typography.Text>
    </Space>
</List>
```
