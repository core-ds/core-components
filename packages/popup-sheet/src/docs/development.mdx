## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getPopupSheetTestIds`.
Импорт из `@balafla/core-components/popup-sheet/shared`.

Функция возвращает объект:

```jsx
{
    popupSheet: dataTestId,
    closer: `${dataTestId}-closer`,
};
```
