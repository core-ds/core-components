## Примеры

Доступны 144, 128, 96, 80, 64, 48, 24 и произвольный размер компонента.

```jsx live mobileHeight={652}
<Container align={isMobile() ? 'center' : 'left'}>
    <Space direction={isMobile() ? 'vertical' : 'horizontal'} align='center' size={24}>
        <CircularProgressBar value={25} size={144} contentColor='primary' />
        <CircularProgressBar value={25} size={128} contentColor='primary' />
        <CircularProgressBar value={25} size={96} contentColor='primary' />
        <CircularProgressBar value={25} size={80} contentColor='primary' />
        <CircularProgressBar value={25} size={64} contentColor='primary' />
        <CircularProgressBar value={25} size={48} contentColor='primary' />
        <CircularProgressBar value={25} size={24} contentColor='primary' />
    </Space>
</Container>
```

## Атрибуты

У компонента есть два состояния — в процессе заполнения и полностью заполненный.
Можно задать направление заполнения, цвет прогрессбара, наличие желоба и контент для обоих состояний.

```jsx live
<Container align={isMobile() ? 'center' : 'left'}>
    <Space direction='horizontal'>
        <CircularProgressBar
            value={0}
            title='0%'
            size={80}
            subtitle='Caption'
            contentColor='primary'
        />
        <CircularProgressBar
            value={50}
            title='50%'
            size={80}
            subtitle='Caption'
            contentColor='primary'
        />
        <CircularProgressBar
            value={100}
            size={80}
            icon={CheckmarkMIcon}
            contentColor='tertiary'
            completeIconColor='positive'
        />
    </Space>
    <Gap size='m' />
    <Space direction='horizontal'>
        <CircularProgressBar value={0} size={80} icon={CheckmarkMIcon} contentColor='tertiary' />
        <CircularProgressBar value={50} size={80} icon={CheckmarkMIcon} contentColor='tertiary' />
        <CircularProgressBar
            value={100}
            size={80}
            icon={CheckmarkMIcon}
            contentColor='tertiary'
            completeIconColor='primary-inverted'
            fillComplete={true}
        />
    </Space>
    <Gap size='m' />
    <Space direction='horizontal'>
        <CircularProgressBar value={0} title='0%' size={80} contentColor='primary' />
        <CircularProgressBar
            value={50}
            title='50%'
            size={80}
            view='negative'
            contentColor='primary'
        />
        <CircularProgressBar
            value={100}
            size={80}
            titleComplete='100%'
            view='negative'
            completeTextColor='negative'
        />
    </Space>
</Container>
```

## Таймер

Компонент можно превратить в таймер, передав соответствующие параметры.

```jsx live
<Container align={isMobile() ? 'center' : 'left'}>
    <Space direction='horizontal'>
        <CircularProgressBar
            timer={true}
            value={60}
            size={80}
            counting='backward'
            directionType='desc'
            contentColor='primary'
        />
        <CircularProgressBar
            timer={true}
            value={60}
            size={80}
            counting='forward'
            directionType='asc'
            contentColor='primary'
        />
    </Space>
    <Gap size='m' />
    <Space direction='horizontal'>
        <CircularProgressBar
            timer={true}
            value={60}
            size={80}
            counting='forward'
            directionType='desc'
            contentColor='primary'
        />
        <CircularProgressBar
            timer={true}
            value={60}
            size={80}
            counting='backward'
            directionType='asc'
            contentColor='primary'
        />
    </Space>
</Container>
```
