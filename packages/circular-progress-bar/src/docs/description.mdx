## Примеры

Доступны XXL, XL, L, M, S, XS и произвольный размеры компонента.

```jsx live mobileHeight={652}
<Container align={isMobile() ? 'center' : 'left'}>
    <Space direction={isMobile() ? 'vertical' : 'horizontal'} align='center' size={24}>
        <CircularProgressBar value={25} size='xxl' contentColor='primary' />
        <CircularProgressBar value={25} size='xl' contentColor='primary' />
        <CircularProgressBar value={25} size='l' contentColor='primary' />
        <CircularProgressBar value={25} size='m' contentColor='primary' />
        <CircularProgressBar value={25} size='s' contentColor='primary' />
        <CircularProgressBar value={25} size='xs' contentColor='primary' />
    </Space>
</Container>
```

## Атрибуты

У компонента есть два состояния — в процессе заполнения и полностью заполненный. 
Можно задать направление заполнения, цвет прогрессбара, наличие желоба и контент для обоих состояний.

```jsx live
<Container align={isMobile() ? 'center' : 'left'} >
    <Space direction='horizontal'>
        <CircularProgressBar value={0} title='0%' size='l' subtitle='Caption' contentColor='primary' />
        <CircularProgressBar value={50} title='50%' size='l' subtitle='Caption' contentColor='primary'/>
        <CircularProgressBar value={100} size='l' icon={CheckmarkMIcon} contentColor='tertiary' completeIconColor='positive'/>
    </Space>
    <Gap size='m'/>
    <Space direction='horizontal'>
        <CircularProgressBar value={0} size='l' icon={CheckmarkMIcon} contentColor='tertiary' />
        <CircularProgressBar value={50} size='l' icon={CheckmarkMIcon} contentColor='tertiary' />
        <CircularProgressBar 
            value={100} 
            size='l' 
            icon={CheckmarkMIcon} 
            contentColor='tertiary'
            completeIconColor='primary-inverted'
            fillComplete={true}
            />
    </Space>
    <Gap size='m'/>
    <Space direction='horizontal'>
        <CircularProgressBar value={0} title='0%' size='l' contentColor='primary' />
        <CircularProgressBar value={50} title='50%' size='l' view='negative' contentColor='primary'/>
        <CircularProgressBar value={100} size='l' titleComplete='100%' view='negative' completeTextColor='negative'/>
    </Space>
</Container>
```
