```css
@import '@alfalab/core-components/vars';
```

Все CSS-переменные можно также импортировать в виде JS-переменных:

```typescript
import * as vars from '@alfalab/core-components/vars';

vars.gap2xl === '32px'; //true
```

Для использования в проекте, добавьте один из [подготовленных бандлов](https://github.com/core-ds/core-components/tree/master/packages/vars/src/bundle) продуктов в один из корневых файлов CSS:

```css
/* app.css */
@import '@alfalab/core-components/vars/bundle/click.css';
```

Теперь все переменные и миксины можно использовать в любом месте проекта:

```css
/* some-component.css */
.class {
    padding: var(--gap-16);
}
```
