```css
@import '@alfalab/core-components/vars';
```

Все CSS-переменные можно также экспортировать в виде JS-переменных:

```typescript
import * as vars from '@alfalab/core-components/vars';

vars.gap2xl === '32px'; //true
```

Для использования в проекте, подключите весь набор переменных в одном из корневых css файлов:

```css
/* app.css */
@import '@alfalab/core-components/vars/index.css';
```

Теперь все переменные и миксины можно использовать в любом месте проекта:

```css
/* some-component.css */
.class {
    padding: var(--gap-m);
}
```
