## Виды бейджей

### Бейдж

`view="count"`

Счётчик действий, уведомлений или сообщений. Если `content` не передан, то отображается точка.
Если `content` больше 99, то отображается 99+.

```jsx live
<Container>
    <Row style={{ padding: 12, backgroundColor: 'rgb(22%, 47%, 98%, 0.1)', borderRadius: '16px 16px 0 0 ' }}>
        <Col>
            <Badge view='count' />
        </Col>
        <Col>
            <Badge view='count' content={1} />
        </Col>
        <Col>
            <Badge view='count' content={100} />
        </Col>
    </Row>
    <Row style={{ marginTop: 0, padding: 12, backgroundColor: 'rgb(22%, 47%, 98%, 0.1)', borderRadius: '0 0 16px 16px'}}>
        <Col>
            <Badge view='count' visibleIconOutline='true' />
        </Col>
        <Col>
            <Badge view='count' content={1} visibleIconOutline='true'/>
        </Col>
        <Col>
            <Badge view='count' content={100} visibleIconOutline='true'/>
        </Col>
    </Row>
</Container>
```
