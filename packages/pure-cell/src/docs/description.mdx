## Лейаут

Компонент представляет из себя структуру с двумя вертикально или горизонтально расположенными слотами: Graphic и Content. 
Горизонтальные и вертикальные отступы могут принимать предустановленные значения.

```jsx live desktopOnly
render(() => {
    const [graphics, setGraphicsRadioValue] = React.useState('left');

    const onGraphicsRadioChange = React.useCallback((_, payload) => {
        setGraphicsRadioValue(payload.value);
    }, []);

    const styleAddons = {
        width: '40px',
        height: '40px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }

    const styleFooter = {
        width: '263px',
        height: '16px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }

    const styleContent = {
        width: '207px',
        height: '20px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }

    const styleGraphics = {
        width: '64px',
        height: '64px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }

    return (
        <Space size={20}>
            <div style={{ width: '340px', backgroundColor: 'rgba(55, 120, 251, 0.1)', padding: '12px 16px' }}>
                <PureCell  padding='default' direction={graphics !== 'top' ? 'horizontal' : 'vertical' }>
                   { graphics !== 'right'  && <PureCell.Graphics verticalAlign='center'>
                        <div style={styleGraphics} />
                    </PureCell.Graphics> }
                    <PureCell.Content>
                        <PureCell.Main>
                            <div style={styleContent} />
                            <div style={styleContent} />
                        </PureCell.Main>
                        <PureCell.Addon verticalAlign='center'>
                            <div style={styleAddons} />
                        </PureCell.Addon>
                        <PureCell.Footer footerPadding='default'>
                            <div style={styleFooter} />
                        </PureCell.Footer>
                    </PureCell.Content>
                    { graphics === 'right' && <PureCell.Graphics>
                        <div style={styleGraphics} />
                    </PureCell.Graphics> }
                </PureCell>
            </div>
            
            <RadioGroup
                direction='vertical'
                name='radioGroup'
                onChange={onGraphicsRadioChange}
                value={graphics}
            >
                <Radio label='Графика слева' value='left' />
                <Radio label='Графика справа' value='right' />
                <Radio label='Графика сверху' value='top' />
            </RadioGroup>
        </Space>
    );
});
```

## .Content

Компонент является внутренней структурой PureCell. Состоит из слотов main (зеленый), addon (фиолетовый) и footer (желтый). 
Каждый из слотов является опциональным. 
Отступы до аддона и футера могут иметь дефолтное значение или отсутствовать.

Контент основных слотов может занимать одну, две или несколько строк. 
Контент правой части не может быть многострочным.
