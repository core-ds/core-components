Компонент текстового ввода даты с указанием времени

## Примеры

### Без пикера
```jsx live
render(
    <div style={{ width: 225 }}>
        <DateTimeInput label="Укажите дату и время" placeholder="ДД.ММ.ГГГГ, ЧЧ:ММ" block />
    </div>
)
```

### С пикером
```jsx live
render(
    <div style={{ width: 230 }}>
        <DateTimeInput label="Укажите дату и время" placeholder="ДД.ММ.ГГГГ, ЧЧ:ММ" picker block/>
    </div>
)
```

### Пикер
Для настроек пикера см. компонент DatePicker
