## Как использовать

Группа радиокнопок используется когда пользователю доступно несколько вариантов выбора.
Обычно от двух до пяти. Если вариантов больше, возможно следует использовать другой контрол,
например [Select](?path=/docs/select--docs).

Для выбора пункта необходимо кликнуть по названию пункта или по радиокнопке. Повторное нажатие не снимет выбор.

Группа может содержать заголовок и подпись.

В группе может быть выбран только один пункт.

Один из пунктов может быть предвыбран по умолчанию, такой пункт необходимо располагать первым в списке.

## Копирайт

Название группы радиокнопок должно быть сформулировано таким образом, чтобы было понятно, что можно выбрать только 1 вариант и
названия вариантов выбора не содержали повторяющуюся часть.

## Горизонтальная группа

Горизонтальное расположение используется при коротких названиях пунктов и малом количестве вариантов — от двух до трех.

```jsx live
render(() => {
    const [value, setValue] = React.useState('one');
    const [error, setError] = React.useState(false);

    const onChange = (_, payload) => {
        setValue(payload.value);
    };

    return (
        <div style={{ width: '100%' }}>
            <RadioGroupDesktop
                label='Ближайшая планета к Солнцу'
                direction='horizontal'
                name='radioGroup'
                onChange={onChange}
                hint='Выберите один из вариантов'
                error={error}
                value={value}
            >
                <Radio label='Меркурий' value='one' size={24} />
                <Radio label='Венера' value='two' size={24} />
                <Radio label='Нептун' value='three' size={24} />
            </RadioGroupDesktop>

            <Gap size='2xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' onClick={() => setError(false)}>
                    По умолчанию
                </Button>
                <Button size='xxs' onClick={() => setError('Правильный ответ: Меркурий')}>
                    Состояние ошибки
                </Button>
            </Space>
        </div>
    );
});
```

## Группа тегов

Теги могут использоваться в качестве радиокнопок в составе группы.
Группа тегов используется при коротких названиях пунктов и малом количестве вариантов — от двух до трех.
Если варианты не помещаются в одну строку они могут перенестись на следующую.

```jsx live
render(() => {
    const [value, setValue] = React.useState('one');
    const [error, setError] = React.useState(false);

    const onChange = (_, payload) => {
        setValue(payload.value);
    };

    return (
        <div style={{ width: '100%' }}>
            <RadioGroupDesktop
                label='Выберите один из вариантов'
                onChange={onChange}
                direction='horizontal'
                type='tag'
                hint='Выберите один из вариантов'
                error={error}
                value={value}
                name='radioGroupTag'
            >
                <Tag value='one'>Вариант 1</Tag>
                <Tag value='two'>Вариант 2</Tag>
                <Tag value='three'>Вариант 3</Tag>
                <Tag value='four'>Вариант 4</Tag>
            </RadioGroupDesktop>

            <Gap size='2xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' onClick={() => setError(false)}>
                    По умолчанию
                </Button>
                <Button size='xxs' onClick={() => setError('Error message')}>
                    Состояние ошибки
                </Button>
            </Space>
        </div>
    );
});
```

## Использование в формах

Не рекомендуется использовать в одной форме несколько вариантов реализации радиогрупп.
Чтобы пользователю было комфортно взаимодействовать с интерфейсом, контролы с одинаковой механикой должны выглядеть одинаково.
