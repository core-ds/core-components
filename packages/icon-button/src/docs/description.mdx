## Виды кнопок

Primary, secondary, tertiary, negative.

```jsx live
<Container>
    <Row>
        {['primary', 'secondary', 'tertiary', 'negative'].map((view) => (
            <Col key={view}>
                <IconButton view={view} size={32} icon={DiamondsMIcon} />
            </Col>
        ))}
    </Row>
</Container>
```

С помощью опции `transparentBg` кнопке можно добавить более явный вид состояний ховера и нажатия.

```jsx live
<Space direction='horizontal' size={8}>
    <IconButton
        view='primary'
        size={32}
        icon={DiamondsMIcon}
    />
    <IconButton
        view='primary'
        size={32}
        icon={DiamondsMIcon}
        transparentBg={true}
    />
</Space>
```

## Размеры компонента

Рекомендуется использовать один из пяти размеров, в зависимости от контекста.

```jsx live
<Space direction='horizontal' size={[20]}>
    {[56, 48, 40, 32, 24].map((size) => (
        <IconButton
            key={size}
            view='primary'
            size={size}
            icon={DiamondsMIcon}
            style={{
                backgroundColor: 'rgba(55, 120, 251, 0.1)',
                borderRadius: 0,
            }}
        />
    ))}
</Space>
```

## Обработка событий

С помощью свойства `loading` можно отобразить состояние загрузки. Минимальное время отображения лоадера — 500мс, чтобы при быстрых ответах от сервера кнопка не «моргала».

```jsx live
render(() => {
    const [loading, setLoading] = React.useState({
        primary: false,
        secondary: false,
    });

    const handleClick = (buttonName, timeout) => {
        setLoading({ ...loading, [buttonName]: true });
        setTimeout(() => {
            setLoading({ ...loading, [buttonName]: false });
        }, timeout);
    };

    return (
        <Container>
            <Row>
                <Col>
                    <IconButton
                        view='primary'
                        size={32}
                        icon={DiamondsMIcon}
                        loading={loading.primary}
                        onClick={() => handleClick('primary', 30)}
                    />
                </Col>
                <Col>
                    <IconButton
                        view='secondary'
                        size={32}
                        icon={DiamondsMIcon}
                        loading={loading.secondary}
                        onClick={() => handleClick('secondary', 1500)}
                    />
                </Col>
            </Row>
        </Container>
    );
});
```
