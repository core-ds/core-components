## Примеры

### Размеры
Рекомендуется использовать S, XS и XXS размеры, в зависимости от контекста.

```jsx live
<Container>
    <Row>
        <Col>
            <IconButton
                view='primary'
                size='s'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ', backgroundColor: 'rgba(55, 120, 251, 0.1)' }}
            />
        </Col>
        <Col>
            <IconButton
                view='primary'
                size='xs'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ', backgroundColor: 'rgba(55, 120, 251, 0.1)' }}
            />
        </Col>
        <Col>
            <IconButton
                view='primary'
                size='xxs'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ', backgroundColor: 'rgba(55, 120, 251, 0.1)' }}
            />
        </Col>
    </Row>
</Container>
```

### Состояние загрузки
С помощью свойства loading можно отобразить состояние загрузки. Минимальное время отображения лоадера - 500мс, чтобы при быстрых ответах от сервера кнопка не «моргала».

```jsx live
render(() => {
    const [loading, setLoading] = React.useState({
        primary: false,
        secondary: false
    });

    const handleClick = (buttonName, timeout) => {
        setLoading({...loading, [buttonName]: true});
        setTimeout(() => {
            setLoading({...loading, [buttonName]: false});
        }, timeout)
    }

    return (
        <Container>
        <Row>
        <Col>
            <IconButton
                view='primary'
                icon={StarMIcon}
                loading={loading.primary}
                onClick={() => handleClick('primary', 1500)}
            />
        </Col>
        <Col>
            <IconButton
                view='secondary'
                icon={StarMIcon}
                loading={loading.secondary}
                onClick={() => handleClick('secondary', 1500)}
            />
        </Col>
        <Col>
             <IconButton
                view='tertiary'
                icon={StarMIcon}
                loading={loading.tertiary}
                onClick={() => handleClick('tertiary', 1500)}
            />
        </Col>
        <Col>
            <IconButton
                view='negative'
                icon={StarMIcon}
                loading={loading.negative}
                onClick={() => handleClick('negative', 1500)}
            />
        </Col> 
        </Row> 
        </Container>
    );
});
```

### Ссылка
Любую кнопку можно превратить в ссылку, передав компоненту пропс `href`.

```jsx live
<Container>
    <IconButton
        view='primary'
        icon={StarMIcon}
        href="https://alfabank.ru"
        target="_blank"
    />
</Container>
```
