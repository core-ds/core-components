## Виды кнопок

Primary, secondary, tertiary, negative.

```jsx live
<Container>
    <Row>
    {['primary', 'secondary','tertiary','negative' ].map(view => (
        <Col>
            <IconButton
                view={view}
                size='xs'
                icon={DiamondsMIcon}
            />
        </Col>
    ))}
    </Row>
</Container>
```

## Размеры

Рекомендуется использовать один из трёх размеров, в зависимости от контекста.

```jsx live
<Container>
    <Row>
    {['s', 'xs','xss' ].map(size => (
        <Col>
            <IconButton
                view='primary'
                size={size}
                icon={DiamondsMIcon}
                style={{
                    backgroundColor: 'rgba(55, 120, 251, 0.1)',
                    borderRadius: 0,
                }}
            />
        </Col>
    ))}
    </Row>
</Container>
```

## Обработка событий

С помощью свойства `loading` можно отобразить состояние загрузки. Минимальное время отображения лоадера — 500мс, чтобы при быстрых ответах от сервера кнопка не «моргала».

```jsx live
render(() => {
    const [loading, setLoading] = React.useState({
        primary: false,
        secondary: false,
    });

    const handleClick = (buttonName, timeout) => {
        setLoading({ ...loading, [buttonName]: true });
        setTimeout(() => {
            setLoading({ ...loading, [buttonName]: false });
        }, timeout);
    };

    return (
        <Container>
            <Row>
                <Col>
                    <IconButton
                        view='primary'
                        size='xs'
                        icon={DiamondsMIcon}
                        loading={loading.primary}
                        onClick={() => handleClick('primary', 30)}
                    />
                </Col>
                <Col>
                    <IconButton
                        view='secondary'
                        size='xs'
                        icon={DiamondsMIcon}
                        loading={loading.secondary}
                        onClick={() => handleClick('secondary', 1500)}
                    />
                </Col>
            </Row>
        </Container>
    );
});
```
