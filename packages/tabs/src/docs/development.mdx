## Подключение

```jsx
// Импорт адаптивных табов
import { TabsResponsive, Tab } from '@alfalab/core-components/tabs';
import { TabsResponsive, Tab } from '@alfalab/core-components/tabs/responsive';

// Импорт только десктопных табов
import { TabsDesktop, Tab } from '@alfalab/core-components/tabs/desktop';

// Импорт только мобильных табов
import { TabsMobile, Tab } from '@alfalab/core-components/tabs/mobile';
```

## Tips & Tricks

- Свойство `title` будет использовано для генерации заголовков, а `id` для переключения табов.

- С помощью свойства `disabled` таб можно заблокировать для выбора.

- По умолчанию в DOM рендерится контент только активного таба, с помощью `keepMounted` можно указать, контент каких табов нужно рендерить в любом случае. Контент в таком случае останется невидимым, но всегда будет находиться в DOM.
