## TabsPrimary

Для переключения контента всей страницы рекомендуется использовать основной вид табов.  
Для TabsPrimary рекомендуется использовать только М размер.

```jsx live
render(() => {
    tabs = [
        { title: 'Aurum', id: 'tab-1' },
        { title: 'Bercelium', id: 'tab-2' },
        { title: 'Curium', id: 'tab-3' },
        { title: 'Neptunium', id: 'tab-4' },
        { title: 'Plutonuim', id: 'tab-5' },
        { title: 'Rubidium', id: 'tab-6' },
        { title: 'Californium', id: 'tab-7' },
    ];

    const [selectedId, setSelectedId] = React.useState('tab-1');
    const [title, setTitle] = React.useState('Aurum');

    const handleChange = (event, { selectedId }) => {
        const currentTab = tabs.filter((tab) => tab.id === selectedId)[0];
        setSelectedId(selectedId);
        setTitle(currentTab.title);
    };

    return (
        <>
            <TabsDesktop selectedId={selectedId} onChange={handleChange}>
                <Tab title='Aurum' id='tab-1' />
                <Tab title='Bercelium' id='tab-2' />
                <Tab title='Curium' id='tab-3' />
                <Tab title='Neptunium' id='tab-4' />
                <Tab title='Plutonuim' id='tab-5' />
                <Tab title='Rubidium' id='tab-6' />
                <Tab title='Californium' id='tab-7' />
            </TabsDesktop>
            <br />
            <div>Выбран таб: {title}</div>
        </>
    );
});
```

## TabsSecondary

Для переключения контента внутри блока рекомендуется использовать второстепенный вид табов.  
Для десктопа рекомендуется использовать S и XS размер табов и XXS размер для мобильной версии интерфейса.

```jsx live
render(() => {
    tabs = [
        { title: 'Aurum', id: 'tab-1' },
        { title: 'Bercelium', id: 'tab-2' },
        { title: 'Curium', id: 'tab-3' },
        { title: 'Neptunium', id: 'tab-4' },
        { title: 'Plutonuim', id: 'tab-5' },
        { title: 'Rubidium', id: 'tab-6' },
        { title: 'Californium', id: 'tab-7' },
    ];

    const [selectSecondaryId, setSelectedSecondaryId] = React.useState('tab-1');
    const [title, setTitle] = React.useState('Aurum');

    const handleSecondaryChange = (event, { selectedId }) => {
        const currentTab = tabs.filter((tab) => tab.id === selectedId)[0];
        setSelectedSecondaryId(selectedId);

    return (
        <div style={{ marginTop: 20 }}>
            <TabsDesktop

        </div>

});
```

## Переполнение

Если табов больше, чем может отобразиться на экране, можно воспользоваться одной из двух опций, горизонтальный скролл или скрытие табов в кнопку `Eщё`.
Для второстепенных табов опция Collapsible недоступна.

```jsx live
const tabs = [
    { title: 'Aurum', id: 'tab-1' },
    { title: 'Bercelium', id: 'tab-2' },
    { title: 'Curium', id: 'tab-3' },
    { title: 'Neptunium', id: 'tab-4' },
    { title: 'Plutonuim', id: 'tab-5' },
    { title: 'Rubidium', id: 'tab-6' },
    { title: 'Californium', id: 'tab-7' },
    { title: 'Einsteinium', id: 'tab-8', rightAddons: <Badge view='count' content={1} /> },
    { title: 'Fermium', id: 'tab-9' },
    { title: 'Mendelevium', id: 'tab-10' },
];
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');
    const [title, setTitle] = React.useState('Aurum');
    const [view, setView] = React.useState('primary');
    const [overflow, setOverflow] = React.useState('scrollable');
    const [collapsibleTabsIds, setCollapsibleTabsIds] = React.useState({ 'tab-1': true });

    const handleChange = (event, { selectedId }) => {
        const currentTab = tabs.filter((tab) => tab.id === selectedId)[0];
        setSelectedId(selectedId);
        setTitle(currentTab.title);
    };
    const handleViewChange = (event, { value }) => setView(value);
    const handleOverflowChange = (event, { value }) => setOverflow(value);

    const handleCollapsedTabsChange = (event, payload) => {
        setCollapsibleTabsIds({ ...collapsibleTabsIds, [payload.name]: payload.checked });
    };

    const manuallyCollapsedTabs = React.useMemo(() => {
        const v = Object.keys(collapsibleTabsIds).filter((k) => !!collapsibleTabsIds[k]);
        return v.length ? v : undefined;
    }, [collapsibleTabsIds]);

    return (
        <div>
            <TabsDesktop
                scrollable={overflow !== 'collapsible'}
                collapsible={overflow !== 'scrollable'}
                collapsedTabsIds={overflow === 'collapsible' && manuallyCollapsedTabs}
                selectedId={selectedId}
                onChange={handleChange}
                view={view}
            >
                {tabs.map((tab) => (
                    <Tab key={tab.id} {...tab} />
                ))}
            </TabsDesktop>
            <br />
            <div>Выбран таб: {title}</div>
            <br />
            <br />
            <div style={{ marginTop: 12 }}>
                <Space size={40}>
                    <RadioGroup
                        name='viewGroup'
                        direction='vertical'
                        label='Тип табов'
                        value={view}
                        onChange={handleViewChange}
                    >
                        <Radio label='Primary' value='primary' />
                        <Radio label='Secondary' value='secondary' />
                    </RadioGroup>
                    <RadioGroup
                        name='componentGroup'
                        direction='vertical'
                        label='Обработка переполнения'
                        value={overflow}
                        onChange={handleOverflowChange}
                    >
                        <Radio label='Scrollable' value='scrollable' />
                        <Radio label='Collapsible' value='collapsible' />
                    </RadioGroup>
                    {overflow === 'collapsible' && (
                        <CheckboxGroup
                            label='Схлопнутые вкладки'
                            name='tabsCheckboxGroup'
                            direction='vertical'
                            value={collapsibleTabsIds}
                            onChange={handleCollapsedTabsChange}
                        >
                            {tabs.map((t) => (
                                <Checkbox
                                    key={t.id}
                                    label={t.title}
                                    name={t.id}
                                    checked={!!collapsibleTabsIds[t.id]}
                                />
                            ))}
                        </CheckboxGroup>
                    )}
                </Space>
            </div>
        </div>
    );
});
```

## Анатомия

С помощью слота `RightAddons` можно кастомизировать тэг. Например, добавить иконку.

```jsx live
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');
    const [selectedSecondaryId, setSelectedSecondaryId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);
    const handleSecondaryChange = (event, { selectedId }) => setSelectedSecondaryId(selectedId);

    const addon = (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '8px',
                height: '24px',
                width: '24px',
                boxSizing: 'border-box',
            }}
        />
    );

    return (
        <>
            <TabsDesktop size='xs' selectedId={selectedId} onChange={handleChange}>
                <Tab title='Aurum' id='tab-1' rightAddons={addon} />
                <Tab title='Bercelium' id='tab-2' rightAddons={addon} />
                <Tab title='Curium' id='tab-3' rightAddons={addon} />
                <Tab title='Neptunium' id='tab-4' rightAddons={addon} />
            </TabsDesktop>
            <div style={{ marginTop: 20 }}>
                <TabsDesktop
                    selectedId={selectedSecondaryId}
                    onChange={handleSecondaryChange}
                    view='secondary'
                    size='xs'
                >
                    <Tab title='Aurum' id='tab-1' rightAddons={addon} />
                    <Tab title='Bercelium' id='tab-2' rightAddons={addon} />
                    <Tab title='Curium' id='tab-3' rightAddons={addon} />
                    <Tab title='Neptunium' id='tab-4' rightAddons={addon} />
                </TabsDesktop>
            </div>
        </>
    );
});
```

## Состояния

Таб может находиться в активном и неактивном состоянии. Взаимодействие может быть ограничено с помощью свойства `disabled`.
Дисейблить активный таб недопустимо.

```jsx live
    const [selectedSecondaryId, setSelectedSecondaryId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);
    const handleSecondaryChange = (event, { selectedId }) => setSelectedSecondaryId(selectedId);

    return (
        <>
            <TabsDesktop size='xs' selectedId={selectedId} onChange={handleChange}>
                <Tab title='Aurum' id='tab-1' />
                <Tab title='Bercelium' id='tab-2' disabled={true} />
                <Tab title='Curium' id='tab-3' />
                <Tab title='Neptunium' id='tab-4' />
                <Tab title='Plutonuim' id='tab-5' />
                <Tab title='Rubidium' id='tab-6' />
                <Tab title='Californium' id='tab-7' />

            <div style={{ marginTop: 20 }}>
                <TabsDesktop
                    selectedId={selectedSecondaryId}
                    onChange={handleSecondaryChange}
                    view='secondary'
                    size='xs'
```
