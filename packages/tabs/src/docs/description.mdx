## Примеры

### Виды табов

У табов есть два вида отображения: primary и secondary

```jsx live
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');
    const [selectSecondaryId, setSelectedSecondaryId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);
    const handleSecondaryChange = (event, { selectedId }) => setSelectedSecondaryId(selectedId);

    return (
        <>
            <TabsResponsive selectedId={selectedId} onChange={handleChange}>
                <Tab title='Aurum' id='tab-1' />
                <Tab title='Bercelium' id='tab-2' />
                <Tab title='Curium' id='tab-3' />
                <Tab title='Neptunium' id='tab-4' />
                <Tab title='Plutonuim' id='tab-5' />
                <Tab title='Rubidium' id='tab-6' />
                <Tab title='Californium' id='tab-7' />
            </TabsResponsive>
            <div style={{ marginTop: 20 }}>
                <TabsResponsive
                    selectedId={selectSecondaryId}
                    onChange={handleSecondaryChange}
                    view='secondary'
                >
                    <Tab title='Aurum' id='tab-1' />
                    <Tab title='Bercelium' id='tab-2' />
                    <Tab title='Curium' id='tab-3' />
                    <Tab title='Neptunium' id='tab-4' />
                    <Tab title='Plutonuim' id='tab-5' />
                    <Tab title='Rubidium' id='tab-6' />
                    <Tab title='Californium' id='tab-7' />
                </TabsResponsive>
            </div>
        </>
    );
});
```

### Адаптивность

-   Можно использовать десктопные, мобильные и адаптивные табы. Адаптивная версия табов перестраивается при 768px.
-   Если табы не помещаются в окне, то можно активировать свойство collapsible, в режиме которого появляется последний таб "Ещё", где размещаются все невлезшие табы на странице.

```jsx live
const tabs = [
    { title: 'Auurum', id: 'tab-1' },
    { title: 'Bercelium', id: 'tab-2' },
    { title: 'Curium', id: 'tab-3' },
    { title: 'Neptunium', id: 'tab-4' },
    { title: 'Plutonuim', id: 'tab-5' },
    { title: 'Rubidium', id: 'tab-6' },
    { title: 'Californium', id: 'tab-7' },
    { title: 'Einsteinium', id: 'tab-8', rightAddons: <Badge view='count' content={1} /> },
    { title: 'Fermium', id: 'tab-9' },
    { title: 'Mendelevium', id: 'tab-10' },
];
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');
    const [view, setView] = React.useState('primary');
    const [width, setWidth] = React.useState('100%');
    const [count, setCount] = React.useState('7');
    const [component, setComponent] = React.useState('TabsResponsive');
    const [collapsible, setCollapsible] = React.useState(false);

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);
    const handleWidthChange = (event, { value }) => setWidth(value);
    const handleCountChange = (event, { value }) => setCount(value);
    const handleViewChange = (event, { value }) => setView(value);
    const handleComponentChange = (event, { value }) => {
        setComponent(value);
        setCollapsible(value === 'TabsCollapsibleDesktop');
    };

    const TabsComponent = React.useMemo(
        () =>
            ({
                TabsDesktop: TabsDesktop,
                TabsMobile: TabsMobile,
                TabsResponsive: TabsResponsive,
                TabsCollapsibleDesktop: TabsDesktop,
            }[component]),
        [component],
    );

    const props = TabsComponent !== TabsDesktop && { fullWidthScroll: true };

    return (
        <>
            <div style={{ width: width }}>
                <TabsComponent
                    {...props}
                    scrollable={true}
                    collapsible={collapsible}
                    selectedId={selectedId}
                    onChange={handleChange}
                    view={view}
                >
                    {tabs.slice(0, +count).map((tab) => (
                        <Tab key={tab.id} {...tab} />
                    ))}
                </TabsComponent>
            </div>
            <div style={{ marginTop: 12 }}>
                <Space size={40}>
                    <RadioGroup
                        name='widthGroup'
                        direction='vertical'
                        label='Контейнер'
                        value={width}
                        onChange={handleWidthChange}
                    >
                        <Radio label='max width' value='100%' />
                        <Radio label='375 px' value='375px' />
                    </RadioGroup>
                    <RadioGroup
                        name='countGroup'
                        direction='vertical'
                        label='Количество табов'
                        value={count}
                        onChange={handleCountChange}
                    >
                        <Radio label='10' value='10' />
                        <Radio label='7' value='7' />
                        <Radio label='2' value='2' />
                    </RadioGroup>
                    <RadioGroup
                        name='viewGroup'
                        direction='vertical'
                        label='Вид'
                        value={view}
                        onChange={handleViewChange}
                    >
                        <Radio label='Primary' value='primary' />
                        <Radio label='Secondary' value='secondary' />
                    </RadioGroup>
                    <RadioGroup
                        name='componentGroup'
                        direction='vertical'
                        label='Адаптивность'
                        value={component}
                        onChange={handleComponentChange}
                    >
                        <Radio label='Desktop' value='TabsDesktop' />
                        <Radio label='Collapsible' value='TabsCollapsibleDesktop' />
                        <Radio label='Mobile' value='TabsMobile' />
                        <Radio label='Responsive' value='TabsResponsive' />
                    </RadioGroup>
                </Space>
            </div>
        </>
    );
});
```

### Tips & Trics

-   Свойство title будет использовано для генерации заголовков, а id для переключения табов.

-   С помощью свойства disabled таб можно заблокировать для выбора.

-   По умолчанию в DOM рендерится контент только активного таба, с помощью keepMounted можно указать, контент каких табов нужно рендерить в любом случае. Контент в таком случае останется невидимым, но всегда будет находиться в DOM.
