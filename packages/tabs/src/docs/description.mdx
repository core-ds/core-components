## Примеры

### Виды табов

У табов есть два вида отображения: primary и secondary

```jsx live
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);

    const element = (
        <TabsResponsive selectedId={selectedId} onChange={handleChange}>
            <Tab title='Aurum' id='tab-1' />
            <Tab title='Bercelium' id='tab-2' />
            <Tab title='Curium' id='tab-3' />
            <Tab title='Neptunium' id='tab-4' />
            <Tab title='Plutonuim' id='tab-5' />
            <Tab title='Rubidium' id='tab-6' />
            <Tab title='Californium' id='tab-7' />
        </TabsResponsive>
    );

    return (
        <>
            {element}
            <div style={{ marginTop: 20 }}>
                {React.cloneElement(element, { view: 'secondary' })}
            </div>
        </>
    );
});
```

### Размеры

Для secondary табов есть возможность управлять размером вложенных компонентов <a href='/docs/компоненты-tag--tag'>Tag</a>.

```jsx live
const sizes = ['s', 'xs', 'xxs'];
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);

    const element = (
        <TabsDesktop selectedId={selectedId} onChange={handleChange} view='secondary'>
            <Tab title='Aurum' id='tab-1' />
            <Tab title='Bercelium' id='tab-2' />
            <Tab title='Curium' id='tab-3' />
            <Tab title='Neptunium' id='tab-4' />
            <Tab title='Plutonuim' id='tab-5' />
            <Tab title='Rubidium' id='tab-6' />
            <Tab title='Californium' id='tab-7' />
        </TabsDesktop>
    );

    return sizes.map((item, index) =>
        index === 0 ? (
            React.cloneElement(element, { key: index, size: item })
        ) : (
            <div style={{ marginTop: 20 }}>
                {React.cloneElement(element, { key: index, size: item })}
            </div>
        ),
    );
});
```

### Слоты

В заголовки табов можно добавлять дополнительные элементы с помощью свойства rightAddons.

```jsx live
render(() => {
    const [selectedId, setSelectedId] = React.useState('tab-1');

    const handleChange = (event, { selectedId }) => setSelectedId(selectedId);

    const element = (
        <TabsResponsive selectedId={selectedId} onChange={handleChange}>
            <Tab title='Aurum' id='tab-1' rightAddons={<StarMIcon />} />
            <Tab title='Bercelium' id='tab-2' rightAddons={<StarMIcon />} />
            <Tab title='Curium' id='tab-3' rightAddons={<StarMIcon />} />
            <Tab title='Neptunium' id='tab-4' rightAddons={<StarMIcon />} />
        </TabsResponsive>
    );

    return (
        <>
            {element}
            <div style={{ marginTop: 20 }}>
                {React.cloneElement(element, { view: 'secondary' })}
            </div>,
        </>
    );
});
```

### Адаптивность

### Tips & Trics

-   Свойство title будет использовано для генерации заголовков, а id для переключения табов.

-   С помощью свойства disabled таб можно заблокировать для выбора.

-   По умолчанию в DOM рендерится контент только активного таба, с помощью keepMounted можно указать, контент каких табов нужно рендерить в любом случае. Контент в таком случае останется невидимым, но всегда будет находиться в DOM.
