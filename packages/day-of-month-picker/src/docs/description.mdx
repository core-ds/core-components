## Выбор дня месяца

```jsx live mobileHeight={640}
render(() => {
    const [value, setValue] = React.useState(1);

    return (
        <>
            <div style={{ width: 320 }}>
                <DayOfMonthPicker value={value} responsive={true} onChange={setValue} />
            </div>
            <p style={{ marginTop: 32, marginBottom: 32 }}>
                Выбран день: {value === 31 ? 'В последний день месяца' : value}
            </p>
        </>
    );
});
//MOBILE
render(() => {
    const [value, setValue] = React.useState(1);

    return (
        <Container>
            <DayOfMonthPickerMobile value={value} onChange={setValue} />
            <p style={{ marginTop: 32, marginBottom: 32 }}>
                Выбран день: {value === 31 ? 'В последний день месяца' : value}
            </p>
        </Container>
    );
});
```

Если выбран `В последний день месяца`, то в качестве значения будет 31

Минимальная допустимая ширина контейнера под календарь — 312px.
