import { Rate } from '@alfalab/core-components-rate';

Компонент для выбора рейтинга (оценки) с помощью иконок.

Используется для:
- оценки товаров, сервисов, контента
- быстрого ввода рейтинга
- отображения текущего рейтинга

Аналоги: Ant Design Rate, Material UI Rating

## Примеры использования

### Базовое использование

Простой рейтинг с значением по умолчанию.

<Rate defaultValue={3} />

### Контролируемый режим

Рейтинг с контролируемым значением.

```tsx
import { useState } from 'react';
import { Rate } from '@alfalab/core-components-rate';

const [value, setValue] = useState(3);

<Rate value={value} onChange={setValue} />
```

### Половинчатые значения

Поддержка половинчатых значений (0.5, 1.5, и т.д.).

<Rate allowHalf defaultValue={2.5} />

### Кастомный символ

Использование кастомного символа вместо звезды.

<Rate character="🔥" defaultValue={4} />

### Подсказки

Добавление подсказок для каждого элемента рейтинга.

<Rate tooltips={['bad', 'ok', 'good', 'great', 'perfect']} defaultValue={3} />

### Отключенное состояние

Рейтинг в отключенном состоянии.

<Rate disabled defaultValue={4} />

### Только для чтения

Рейтинг в режиме только для чтения.

<Rate readOnly defaultValue={4} />

### Размеры

Компонент поддерживает три размера: s, m, l.

<div style={{ display: 'flex', flexDirection: 'column', gap: 16 }}>
    <Rate size="s" defaultValue={3} />
    <Rate size="m" defaultValue={3} />
    <Rate size="l" defaultValue={3} />
</div>

### Без сброса

Отключение сброса значения по повторному клику.

<Rate allowClear={false} defaultValue={3} />

### Кастомное количество элементов

Изменение количества элементов рейтинга.

<Rate count={10} defaultValue={7} />
