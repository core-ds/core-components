## Базовое использование

Простой рейтинг с значением по умолчанию.

```jsx live
render(() => <Rate defaultValue={3} />);
```

## Контролируемый режим

Рейтинг с контролируемым значением.

```jsx live
render(() => {
    const [value, setValue] = React.useState(3);

    return <Rate value={value} onChange={setValue} />;
});
```

## Половинчатые значения

Поддержка половинчатых значений (0.5, 1.5, и т.д.).

```jsx live
render(() => <Rate allowHalf={true} defaultValue={2.5} />);
```

## Кастомный символ

Использование кастомного символа вместо звезды.

```jsx live
render(() => <Rate character='🔥' count={5} defaultValue={4} />);
```

## Подсказки

Добавление подсказок для каждого элемента рейтинга.

```jsx live
render(() => (
    <Rate 
        tooltips={['bad', 'ok', 'good', 'great', 'perfect']} 
        count={5}
        defaultValue={3} 
    />
));
```

## Отключенное состояние

Рейтинг в отключенном состоянии.

```jsx live
render(() => <Rate disabled={true} count={5} defaultValue={4} />);
```

## Только для чтения

Рейтинг в режиме только для чтения.

```jsx live
render(() => <Rate readOnly={true} count={5} defaultValue={4} />);
```

## Размеры

Компонент поддерживает три размера: s, m, l.

```jsx live
render(() => (
    <Space direction='vertical' gap={16}>
        <Rate size='s' count={5} defaultValue={3} />
        <Rate size='m' count={5} defaultValue={3} />
        <Rate size='l' count={5} defaultValue={3} />
    </Space>
));
```

## Без сброса

Отключение сброса значения по повторному клику.

```jsx live
render(() => <Rate allowClear={false} count={5} defaultValue={3} />);
```

## Кастомное количество элементов

Изменение количества элементов рейтинга.

```jsx live
render(() => <Rate count={10} defaultValue={7} />);
```
