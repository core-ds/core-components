## Стили текста

Компоненту можно задать необходимый стиль текста.

```jsx live
<div>
    <TextComponent view='headline-system-small'>Headline-system-small</TextComponent>
    <Gap size={16} />
    <TextComponent view='paragraph-primary-medium'>Paragraph-primary-medium</TextComponent>
</div>
```

## Цвет текста и фона

Тексту и фону под ним можно здать необходимый цвет.

```jsx live
<TextComponent
    color='var(--color-light-text-positive)'
    textBackgroundColor='var(--color-light-status-muted-attention)'
>
    Пример с покраской фона
</TextComponent>
```

## Отсечение текста

Можно настроить отсечение текста при переполнении одной, двух или трёх строк.

```jsx live
<div style={{ width: isMobile() ? 320 : 430 }}>
    <TextComponent rowLimit={1}>
        Я взял со стола, как теперь помню, червонного туза и бросил кверху: дыхание у всех
        остановилось;
    </TextComponent>
</div>
```

## Выравнивание

```jsx live
<div style={{ width: isMobile() ? 320 : 200 }}>
    <TextComponent tag='div' align='center'>
        По центру
    </TextComponent>
    <Gap size='s' />
    <TextComponent tag='div' align='left'>
        По левому краю
    </TextComponent>
    <Gap size='s' />
    <TextComponent tag='div' align='right'>
        По правому краю
    </TextComponent>
</div>
```
