## Пример

В этом примере при изменении ширины окна браузера кнопка будет переключаться между мобильной и десктопной версией.

```jsx live
<div>
    <Mq query='--mobile'>
        <Button breakpoint={BREAKPOINT} >Мобильная кнопка</Button>
    </Mq>
    <Mq query='--tablet-s'>
        <Button breakpoint={BREAKPOINT} >Десктопная кнопка</Button>
    </Mq>
</div>
```
