## Размеры

Основным типом инпутов в десктопном вебе являются инпуты размера M с внутренним расположением лейбла. 
Для мобильного веба рекомендуемый размер S с внешним расположением лейбла.

```jsx live
render(() => {
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <Input 
                placeholder='Введите текст' 
                block={true} 
                label='Текст' 
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint} 
            />
            <Gap size='m' />
            <Input 
                placeholder='Введите email' 
                block={true} 
                label='Email' 
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint} 
            />
        </div>
    );
});
```

## Валидация

См. гайд [валидация](?path=/docs/guidelines-validation--docs).

```jsx live
render(() => {
    const [error, setError] = React.useState(false);

    return (
        <div style={{ width: 320 }}>

                    block={true}

            <Gap size='xl' />

                Состояние ошибки
            </Button>
        </div>
    );
});
```

## Disabled и ReadOnly

Доступный только для чтения инпут может быть выбран с клавиатуры, его значение передается при отправке формы.
Заблокированный инпут нельзя выбрать с клавиатуры и его значение не будет отправлено.

```jsx live
render(() => {
    const [inputProps, setInputProps] = React.useState({});
    const breakpoint = 600;

    return (
        <div>
            <div style={{ width: 320 }}>
                <Input 
                    block={true} 
                    placeholder='Placeholder' 
                    label='Label' 
                    labelView={isMobile() ? 'outer' : 'inner'}
                    size={isMobile() ? 's' : 'm'}
                    breakpoint={breakpoint}  
                    {...inputProps} 
                />
            </div>
            <Gap size='xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' view='secondary' onClick={() => setInputProps({})}>
                    По умолчанию
                </Button>
                <Button
                    size='xxs'
                    view='secondary'
                    onClick={() => setInputProps({ disabled: true })}
                >
                    Заблокирован
                </Button>
                <Button
                    size='xxs'
                    view='secondary'
                    onClick={() => setInputProps({ readOnly: true })}
                >
                    Только чтение
                </Button>
            </Space>
        </div>
    );
});
```

## Дополнительная информация

Для упрощения пользовательского ввода используется несколько паттернов вывода дополнительной информации.

1. Placeholder — подсказка внутри поля, отображается до тех пор, пока пользователь не начнет вводить значение.
2. Hint — подсказка под полем. Если поле перейдёт в состоянии ошибки hint будет заменен на сообщение об ошибке.
3. Tooltip — иконка с подсказкой.

```jsx live
render(() => {
    const [open, setOpen] = React.useState(false);
    const breakpoint = 600;

    /* Указать для IconButton через className */
    const css = `
        button[data-test-id='icon'] {
            margin-right: var(--gap-s-neg);
        }
    `;

    return (
        <div style={{ width: 320 }}>
            <style>{css}</style>
            <Input
                block={true}
                label={!isMobile() && 'Инпут с плейсхолдером'}
                placeholder={isMobile() ? 'Инпут с плейсхолдером' : 'Placeholder'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                hint='Подсказка под полем'
                rightAddons={
                    <Tooltip
                        open={open}
                        targetClassName='target'
                        content={
                            <span style={{ whiteSpace: 'pre-wrap' }} >
                                {`Сообщение тултипа\nотображается при\n${isMobile() ? 'клике' : 'наведении'}`}
                            </span>
                        }
                        onOpen={() => setOpen(true)}
                        onClose={() => setOpen(false)}
                        position='right'
                    >
                        <IconButton
                            view='secondary'
                            size='s'
                            icon={InformationCircleMIcon}
                            dataTestId='icon'
                        />
                    </Tooltip>
                }
            />
        </div>
    );
});
```

## Статусы

В некоторых кейсах может потребоваться дополнительная индикация ошибки или успеха.

```jsx live
render(() => {
    const [success, setSuccess] = React.useState(false);
    const [value, setValue] = React.useState('');

    /* Не показываем вторую иконку ошибки в теме сайта */
    const css = `
        :root {
            --input-error-icon-display: none;
            --input-error-icon-not-alone-display: none;
        }
    `;

    const handleChange = (e) => {
        const value = e.target.value;
        setValue(value);

        if (value.length === 5) {
            setSuccess(true);
        } else {
            setSuccess(false);
        }
    };

    const hasError = value !== '' && !success;
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <style>{css}</style>
            <Input
                value={value}
                block={true}
                onChange={handleChange}
                label='Введите 5 символов'
                placeholder='Placeholder'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                success={success}
                error={hasError}
                rightAddons={
                    hasError ? (
                        <Badge
                            view='icon'
                            size='m'
                            iconColor='negative'
                            content={
                                <ExclamationCircleMIcon
                                    style={{ color: 'var(--color-light-graphic-negative)' }}
                                />
                            }
                        />
                    ) : undefined
                }
            />
        </div>
    );
});
```

## Крестик очистки

В редких кейсах может потребоваться механика быстрой очистки введенного контента.
Рекомендуется использовать данную механику точечно, а не глобально ко всем полям большой формы.

```jsx live
render(() => {
    const [value, setValue] = React.useState('Value');
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <Input
                value={value}
                block={true}
                onChange={(e) => setValue(e.target.value)}
                label='Label'
                placeholder='Placeholder'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                clear={true}
                onClear={() => setValue('')}
            />
        </div>
    );
});
```

## Кликабельные аддоны

Для кликабельных адоннов рекомендуем использовать увеличенную область нажатия.

```jsx live
render(() => {
    const [isVisible, setIsVisible] = React.useState(false);
    const [title, setTitle] = React.useState('');
    const [valueOne, setValueOne] = React.useState('Value');
    const [valueTwo, setValueTwo] = React.useState('Value');

    const toggleVisiblity = () => setIsVisible((prev) => !prev);

    const singleIcon = () => {
        setTitle('Вы восхитительны!');
        toggleVisiblity();
    };

    const leftIcon = () => {
        setTitle('Левая');
        toggleVisiblity();
    };

    const rightIcon = () => {
        setTitle('Правая');
        toggleVisiblity();
    };

    const hideNotification = React.useCallback(() => setIsVisible(false), []);

    const styleWrapperIcons = {
        height: '100%',
        display: 'flex',
        gap: 4,
        marginRight: 'var(--gap-xs-neg)',
    };

    /* Указать для IconButton через className */
    const css = `
        button[data-test-id='icon'] {
            margin-right: var(--gap-s-neg);
        }
    `;

    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <style>{css}</style>
            <Input
                value={valueOne}
                block={true}
                onChange={(e) => setValueOne(e.target.value)}
                label='Label'
                placeholder='Placeholder'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                rightAddons={
                    <IconButton
                        view='secondary'
                        onClick={singleIcon}
                        icon={DiamondsMIcon}
                        size='s'
                        dataTestId='icon'
                    />
                }
            />
            <Gap size='l' />
            <Input
                value={valueTwo}
                block={true}
                onChange={(e) => setValueTwo(e.target.value)}
                label='Label'
                placeholder='Placeholder'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                rightAddons={
                    <div style={styleWrapperIcons}>
                        <IconButton
                            view='secondary'
                            onClick={leftIcon}
                            icon={DiamondsMIcon}
                            size='xs'
                            alignIcon='right'
                        />
                        <IconButton
                            view='secondary'
                            onClick={rightIcon}
                            icon={FlowerMIcon}
                            size='xs'
                            alignIcon='left'
                        />
                    </div>
                }
            />
            <Toast
                title={title}
                open={isVisible}
                onClose={hideNotification}
                hasCloser={false}
                autoCloseDelay={2000}
                style={{ left: isMobile() ? '30%' : '41%' }}
            />
        </div>
    );
});
```
