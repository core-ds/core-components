## Форма и размеры

### SuperEllipse

128px, 80px, 64px, 48px, 40px, 24px.

```jsx live
const SIZES = ['128', '80', '64', '48', '40', '24'];

render(
    <Container>
        <Row>
            {SIZES.map(size => (
                <Col key={size}>
                    <SuperEllipse size={size}>
                        <SfTouchIdXxlIcon />
                    </SuperEllipse>
                </Col>
            ))}
        </Row>
    </Container>,
);
```

## Фон

Можно передавать заливку и изображение, включать и выключать обводку.

```jsx live
<Container>
    <Row>
        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor='transparent'
                border={true}
            />
        </Col>

        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor="#EBF9F1"
                border={true}
            />
        </Col>

        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor="#EBF9F1"
            />
        </Col>
    </Row>
</Container>
```

## Контент

```jsx live

    <Row>
        <Col>
            <SuperEllipse size='80'>

            </SuperEllipse>

        </Col>

```

## Аддоны

Без, правый верхний, правый нижний, оба.

```jsx live
const badge = <Badge
    view='icon'
    size='m'
    iconColor='positive'
    visibleColorOutline={true}
    content={<CheckmarkCircleMIcon />}
/>;

render(
    <Container>
        <Row>
            <Col>
                <SuperEllipse size='80' />
            </Col>

            <Col>
                <SuperEllipse size='80' topAddons={badge} />
            </Col>

            <Col>
                <SuperEllipse size='80' bottomAddons={badge} />
            </Col>

            <Col>
                <SuperEllipse
                    size='80'
                    topAddons={badge}
                    bottomAddons={badge}
                />
            </Col>
        </Row>
    </Container>,
);
```
