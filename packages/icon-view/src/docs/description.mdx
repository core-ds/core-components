## Форма и размеры

### SuperEllipse

128px, 80px, 64px, 48px, 40px.

```jsx live
const SIZES = ['128', '80', '64', '48', '40'];

render(
    <Container>
        <Row>
            {SIZES.map(size => (
                <Col key={size}>
                    <SuperEllipse size={size}>
                        <SfTouchIdXxlIcon />
                    </SuperEllipse>
                </Col>
            ))}
        </Row>
    </Container>,
);
```

### Circle

80px, 64px, 48px, 40px.

```jsx live
const SIZES = ['80', '64', '48', '40'];

render(
    <Container>
        <Row>
            {SIZES.map(size => (
                <Col key={size}>
                    <Circle size={size}>
                        <SfTouchIdXxlIcon />
                    </Circle>
                </Col>
            ))}
        </Row>
    </Container>,
);
```

## Фон

Можно передавать заливку и изображение, включать и выключать обводку.

```jsx live
<Container>
    <Row>
        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor='transparent'
                border={true}
            />
        </Col>

        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor="#EBF9F1"
                border={true}
            />
        </Col>

        <Col>
            <SuperEllipse
                size='80'
                imageUrl="./images/smile.png"
                backgroundColor="#EBF9F1"
            />
        </Col>
    </Row>
</Container>
```

## Контент

Иконка, инициалы, без контента

```jsx live
<Container>
    <Row>
        <Col>
            <SuperEllipse size='80'>
                <SfTouchIdXxlIcon />
            </SuperEllipse>
        </Col>

        <Col>
            <SuperEllipse size='80'>
                <Typography.Title view='small' weight='bold'>
                    ЙО
                </Typography.Title>
            </SuperEllipse>
        </Col>

        <Col>
            <SuperEllipse size='80' />
        </Col>
    </Row>
</Container>
```

## Аддоны

Без, правый верхний, правый нижний, оба.

```jsx live
<Container>
    <Row>
        <Col>
            <SuperEllipse size='80' />
        </Col>

        <Col>
            <SuperEllipse size='80' topAddons={<CheckmarkCircleMIcon color='#2fc26e' />} />
        </Col>

        <Col>
            <SuperEllipse size='80' bottomAddons={<CheckmarkCircleMIcon color='#2fc26e' />} />
        </Col>

        <Col>
            <SuperEllipse
                size='80'
                topAddons={<CheckmarkCircleMIcon color='#2fc26e' />}
                bottomAddons={<CheckmarkCircleMIcon color='#2fc26e' />}
            />
        </Col>
    </Row>
</Container>
```
