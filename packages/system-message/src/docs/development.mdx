## Подключение

Компаунд

```jsx
// Респонсив версия
import { SystemMessage } from '@alfalab/core-components/system-message';

// Десктоп версия
import { SystemMessageDesktop } from '@alfalab/core-components/system-message/desktop';

// Мобильная версия
import { SystemMessageMobile } from '@alfalab/core-components/system-message/mobile';
```

Атомы

```jsx
// Респонсив версия
import {
    SystemMessageComponent,
    SystemMessageCaption,
    SystemMessageControls,
    SystemMessageGraphic,
    SystemMessageSubtitle,
    SystemMessageTitle,
} from '@alfalab/core-components/system-message';

// Десктоп версия
import {
    SystemMessageComponentDesktop,
    SystemMessageCaptionDesktop,
    SystemMessageControlsDesktop,
    SystemMessageGraphicDesktop,
    SystemMessageSubtitleDesktop,
    SystemMessageTitleDesktop,
} from '@alfalab/core-components/system-message/desktop';

// Мобильная версия
import {
    SystemMessageComponentMobile,
    SystemMessageCaptionMobile,
    SystemMessageControlsMobile,
    SystemMessageGraphicMobile,
    SystemMessageSubtitleMobile,
    SystemMessageTitleMobile,
} from '@alfalab/core-components/system-message/mobile';
```

## Использование dataTestId

В компоненте используются модификаторы для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getSystemMessageTestIds`.
Импорт из `@alfalab/core-components/system-message/shared`.

Функция возвращает объект:

```jsx
{
    systemMessage: dataTestId,
    caption: `${dataTestId}-caption`,
    controls: `${dataTestId}-controls`,
    graphic: `${dataTestId}-graphic`,
    subtitle: `${dataTestId}-subtitle`,
    title: `${dataTestId}-core-title`,
```
