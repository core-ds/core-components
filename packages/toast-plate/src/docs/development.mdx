## Подключение

```jsx
import { ToastPlate } from '@balafla/core-components/toast-plate';
import { ToastPlateMobile } from '@balafla/core-components/toast-plate/mobile';
import { ToastPlateDesktop } from '@balafla/core-components/toast-plate/desktop';
```

Из индекса импортируется responsive версия компонента.

## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.\
Для удобного поиска элементов можно воспользоваться функцией `getBaseToastPlateTestIds`.\
Импорт из `@balafla/core-components/toast-plate/shared`.

Функция возвращает объект:

```jsx
{
    badge: `${dataTestId}-badge`,,
    component: `${dataTestId}-component`,
};
```
