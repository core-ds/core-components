## Обычная ссылка

Нестареющая классика. Ссылка может вести как на страницу внутри системы, так и на внешний ресурс.

```jsx live
<Space>
    <Typography.Text view='primary-medium'>
        <Link
            view='default'
            rel='noopener'
            target='_blank'
            href='https://alfabank.ru/get-money/credit-cards/100-days/'
        >
            Default
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link
            view='primary'
            rel='noopener'
            target='_blank'
            href='https://alfabank.ru/get-money/credit-cards/100-days/'
        >
            Primary
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link
            view='secondary'
            rel='noopener'
            target='_blank'
            href='https://alfabank.ru/get-money/credit-cards/100-days/'
        >
            Secondary
        </Link>
    </Typography.Text>
</Space>
```

## Псевдоссылка

Вызывает информационный слой (collapse, dropdown, popup) без перехода на другую страницу.

```jsx live
<Space>
    <Typography.Text view='primary-medium'>
        <Link pseudo={true} view='default'>
            Default
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link pseudo={true} view='primary'>
            Primary
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link pseudo={true} view='secondary'>
            Secondary
        </Link>
    </Typography.Text>
</Space>
```

## Подчёркивание ссылок

Подчёркивание ссылок не является обязательным, но помогает пользователю определить их тип.

```jsx live
<Space>
    <Typography.Text view='primary-medium'>
        <Link view='default'>Default link</Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link pseudo={true} view='default'>
            Pseudo link
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link underline={false} view='default'>
            Link without underline
        </Link>
    </Typography.Text>
</Space>
```

## Анатомия

С помощью слотов `leftAddons` и `rightAddons` можно кастомизировать ссылку. Например, добавить иконку.
Переданный контент будет отрисован слева или справа от текста ссылки.

```jsx live
<Space>
    <Typography.Text view='primary-medium'>
        <Link view='default' leftAddons={<StarMIcon />}>
            Default link
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link view='default' rightAddons={<StarMIcon />}>
            Default link
        </Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link view='default' leftAddons={<StarMIcon />} rightAddons={<StarMIcon />}>
            Default link
        </Link>
    </Typography.Text>
</Space>
```

## Размеры

Cсылка наследует текстовый стиль от родителя.

```jsx live
<Space>
    <Typography.Text view='primary-large'>
        <Link view='default'>Paragraph primary large</Link>
    </Typography.Text>
    <Typography.Text view='primary-medium'>
        <Link view='default'>Paragraph primary medium</Link>
    </Typography.Text>
    <Typography.Text view='primary-small'>
        <Link view='default'>Paragraph primary small</Link>
    </Typography.Text>
</Space>
```
