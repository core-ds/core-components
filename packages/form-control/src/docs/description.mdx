## Лейбл

Лейбл может быть размещен внутри либо снаружи инпута или отсутствовать.

```jsx live
    <div style={{ width: 320 }}>
        <Input
            block
            placeholder='Placeholder'
            label='Лейбл внутри'
            size={isMobile() ? 48 : 56}
            breakpoint={BREAKPOINT}
        />
        <Gap size={20} />
        <Input
            block
            placeholder='Placeholder'
            label='Лейбл снаружи'
            size={isMobile() ? 48 : 56}
            labelView='outer'
            breakpoint={BREAKPOINT}
        />
        <Gap size={20} />
        <Input
            block
            placeholder='Placeholder'
            size={isMobile() ? 48 : 56}
            breakpoint={BREAKPOINT}
        />
    </div>
```

## Префиксы и суффиксы

Инпут может содержать предзаполненное значение, располагающееся спереди или сзади вводимого пользователем значения.

```jsx live
render(() => {
    const SuffixInputComponent = isMobile() ? withSuffix(InputMobile) : withSuffix(InputDesktop);

    return (
        <div style={{ width: 320 }}>
            <SuffixInputComponent
                value='Value'
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                suffix={<span style={{ color: 'var(--color-light-text-secondary)' }}> Suffix</span>}
                breakpoint={BREAKPOINT}
            />
        </div>
    );
});
```

## Аддоны

С помощью `leftAddons` и `rightAddons` можно кастомизировать инпут. Каждый аддон может содержать до двух элементов.
Тип, приоритет расположения контента в слотах и механики взаимодействия описаны в спецификациях к конкретным типам инпутов.

```jsx live
render(() => {
    const styleWrapperIcons = {
        height: '100%',
        display: 'flex',
        gap: 4,
    };

    return (
        <div style={{ width: 320 }}>
            <FormControl
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 48 : 56}
                breakpoint={BREAKPOINT}
                leftAddons={
                    <IconButton
                        view='secondary'
                        icon={DiamondsMIcon}
                        size={32}
                        alignIcon='left'
                    />
                }
            />
            <Gap size={20} />
            <FormControl
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 48 : 56}
                breakpoint={BREAKPOINT}
                rightAddons={
                    <IconButton
                        view='secondary'
                        icon={DiamondsMIcon}
                        size={32}
                        alignIcon='right'
                    />
                }
            />
            <Gap size={20} />
            <FormControl
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 48 : 56}
                breakpoint={BREAKPOINT}
                leftAddons={
                    <div style={{...styleWrapperIcons, marginLeft: 'var(--gap-8-neg)' }}>
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size={32}
                            alignIcon='right'
                        />
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size={32}
                            alignIcon='left'
                        />
                    </div>
                }
                rightAddons={
                    <div style={{...styleWrapperIcons, marginRight: 'var(--gap-8-neg)' }}>
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size={32}
                            alignIcon='right'
                        />
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size={32}
                            alignIcon='left'
                        />
                    </div>
                }
            />
        </div>
    );
});
```

## BottomAddon

При необходимости в BottomAddon может быть передан любой контент, например шкала для слайдера.

```jsx live
render(() => {
    const [value, setValue] = React.useState(3);

    return (
        <div style={{ width: 320 }}>
            <SliderInput
                value={value}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                pips={{
                    mode: 'values',
                    values: [1, 10 / 2, 10],
                }}
                min={1}
                max={10}
                step={1}
                block={true}
                onChange={(_, { value }) => setValue(value)}
                breakpoint={BREAKPOINT}
            />
        </div>
    );
});
```
