## Размеры

XL (72px), L (64px), M (56px), S (48px).

```jsx live
render(() => {
    return (
        <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
            <FormControl block={true} label='Label' size='xl' />
            <Gap size='m' />
            <FormControl block={true} label='Label' size='l' />
            <Gap size='m' />
            <FormControl block={true} label='Label' size='m' />
            <Gap size='m' />
            <FormControl block={true} label='Label' size='s' />
        </div>
    );
});
```

## Лейбл

Лейбл может быть размещен внутри либо снаружи инпута или отсутствовать.

```jsx live
render(() => {
    return (
        <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
            <Input block label='Лейбл внутри' size='m' />
            <Gap size='m' />
            <Input
                block
                placeholder='Placeholder'
                label='Лейбл снаружи'
                size='m'
                labelView='outer'
            />
            <Gap size='m' />
            <Input block placeholder='Placeholder' size='m' labelView='outer' />
        </div>
    );
});
```

## Аддоны

С помощью `leftAddons` и `rightAddons` можно кастомизировать инпут. Каждый аддон может содержать до двух элементов.
Тип, приоритет расположения контента в слотах и механики взаимодействия описаны в спецификациях к конкретным типам инпутов.

```jsx live
render(() => {
    return (
        <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
            <FormControl block={true} label='Label' size='m' leftAddons={<StarMIcon />} />
            <Gap size='m' />
            <FormControl block={true} label='Label' size='m' rightAddons={<StarMIcon />} />
            <Gap size='m' />
            <FormControl
                block={true}
                label='Label'
                size='m'
                leftAddons={
                    <div>
                        <StarMIcon />
                        <Gap size='xxs' direction='horizontal' />
                        <StarMIcon />
                    </div>
                }
                rightAddons={
                    <div>
                        <StarMIcon />
                        <Gap size='xxs' direction='horizontal' />
                        <StarMIcon />
                    </div>
                }
            />
        </div>
    );
});
```

## Суффиксы

Инпут может содержать предзаполненное значение, располагающееся спереди вводимого пользователем значения.

```jsx live
render(() => {
    const SuffixInput = withSuffix(Input);

    return (
        <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
            <SuffixInput
                value='Value'
                block={true}
                label='Label'
                size='m'
                suffix={<span style={{ color: 'var(--color-light-text-secondary)' }}> Suffix</span>}
            />
        </div>
    );
});
```

## Хинт

Это тоже слот. При необходимости в него может быть передан любой контент, например шкала для слайдера.

```jsx live
render(() => {
    return (
        <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
            <FormControl block={true} label='Label' hint='Hint' size='m' />
        </div>
    );
});
```
