## Размеры

Компонент доступен в размерах XL, L, M, S.

```jsx live
render(() => {
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'} 
                size='xl' 
                breakpoint={breakpoint}
            />
            <Gap size='m' />
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'} 
                size='l' 
                breakpoint={breakpoint}
            />
            <Gap size='m' />
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'} 
                size='m' 
                breakpoint={breakpoint}
            />
            <Gap size='m' />
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'} 
                size='s' 
                breakpoint={breakpoint}
            />
        </div>
    );
});
```

## Лейбл

Лейбл может быть размещен внутри либо снаружи инпута или отсутствовать.

```jsx live
render(() => {
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <Input 
                block
                placeholder='Placeholder'
                label='Лейбл внутри'
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
            />
            <Gap size='l' />
            <Input
                block
                placeholder='Placeholder'
                label='Лейбл снаружи'
                size={isMobile() ? 's' : 'm'}
                labelView='outer'
                breakpoint={breakpoint}
            />
            <Gap size='l' />
            <Input 
                block 
                placeholder='Placeholder' 
                size={isMobile() ? 's' : 'm'} 
                breakpoint={breakpoint} 
            />
        </div>
    );
});
```

## Префиксы и суффиксы

Инпут может содержать предзаполненное значение, располагающееся спереди или сзади вводимого пользователем значения. 

```jsx live
render(() => {
    const SuffixInputComponent = isMobile() ? withSuffix(InputMobile) : withSuffix(InputDesktop);
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <SuffixInputComponent
                value='Value'
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                suffix={<span style={{ color: 'var(--color-light-text-secondary)' }}> Suffix</span>}
                breakpoint={breakpoint}
            />
        </div>
    );
});
```

## Хинт

Под полем ввода можно разместить подсказку.

```jsx live
render(() => {
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>

                block={true} 
                label='Label' 
                hint='Hint' 
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
            />
        </div>
    );

```

## Аддоны

С помощью `leftAddons` и `rightAddons` можно кастомизировать инпут. Каждый аддон может содержать до двух элементов.
Тип, приоритет расположения контента в слотах и механики взаимодействия описаны в спецификациях к конкретным типам инпутов.

```jsx live
render(() => {
    const styleWrapperIcons = {
        height: '100%',
        display: 'flex',
        gap: 4,
    };

    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint} 
                leftAddons={
                    <IconButton
                        view='secondary'
                        icon={DiamondsMIcon}
                        size='xs'
                        alignIcon='left'
                    />
                } 
            />
            <Gap size='l' />
            <FormControl 
                block={true} 
                label='Label' 
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint} 
                rightAddons={
                    <IconButton
                        view='secondary'
                        icon={DiamondsMIcon}
                        size='xs'
                        alignIcon='right'
                    />
                } 
            />
            <Gap size='l' />
            <FormControl
                block={true}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                breakpoint={breakpoint}
                leftAddons={
                    <div style={{...styleWrapperIcons, marginLeft: 'var(--gap-xs-neg)' }}>
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size='xs'
                            alignIcon='right'
                        />
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size='xs'
                            alignIcon='left'
                        />
                    </div>
                }
                rightAddons={
                    <div style={{...styleWrapperIcons, marginRight: 'var(--gap-xs-neg)' }}>
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size='xs'
                            alignIcon='right'
                        />
                        <IconButton
                            view='secondary'
                            icon={DiamondsMIcon}
                            size='xs'
                            alignIcon='left'
                        />
                    </div>
                }
            />
        </div>
    );
});
```

## BottomAddon

При необходимости в BottomAddon может быть передан любой контент, например шкала для слайдера.

```jsx live
render(() => {
    const [value, setValue] = React.useState(3);
    const breakpoint = 600;

    return (
        <div style={{ width: 320 }}>
            <SliderInput
                value={value}
                label='Label'
                labelView={isMobile() ? 'outer' : 'inner'}
                size={isMobile() ? 's' : 'm'}
                pips={{
                    mode: 'values',
                    values: [1, 10 / 2, 10],
                }}
                min={1}
                max={10}
                step={1}
                block={true}
                onChange={(_, { value }) => setValue(value)}
                breakpoint={breakpoint}
            />
        </div>
    );
});
```
