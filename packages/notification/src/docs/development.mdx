## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.\
Для удобного поиска элементов можно воспользоваться функцией `getNotificationTestIds`.\
Импорт из `@balafla/core-components/notification/shared`.

Функция возвращает объект:

```jsx
{
    badge: `${dataTestId}-badge`,,
    component: `${dataTestId}-component`,
};
```
