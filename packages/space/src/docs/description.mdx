## Примеры

Фиксированное расстояние между блоками.

```jsx live
render(() => {
    const styleBlock = {
        width: '64px',
        height: '64px',
        borderRadius: '8px',
        backgroundColor: 'var(--color-light-specialbg-secondary-transparent)',
    };
    return (
        <Space direction='horizontal' align={isMobile() ? 'center' : 'start'}>
            {new Array(3).fill(null).map((_, idx) => (
                <div key={idx} style={styleBlock} />
            ))}
        </Space>
    );
});
```

Использование разделителя между блоками.

```jsx live
render(() => {
    const styleBlock = {
        width: '64px',
        height: '64px',
        borderRadius: '8px',
        backgroundColor: 'var(--color-light-specialbg-secondary-transparent)',
    };
    return (
        <Space divider={<Divider />} align={isMobile() ? 'center' : 'start'}>
            {new Array(4).fill(null).map((_, idx) => (
                <div key={idx} style={styleBlock} />
            ))}
        </Space>
    );
});
```

Автоперенос строки с заданными вертикальными и горизонтальными расстояниями между блоками.

```jsx live
render(() => {
    const styleBlock = {
        width: '64px',
        height: '64px',
        borderRadius: '8px',
        backgroundColor: 'var(--color-light-specialbg-secondary-transparent)',
    };
    return (
        <Space direction='horizontal' size={[16, 20]} wrap>
            {new Array(20).fill(null).map((_, idx) => (
                <div key={idx} style={styleBlock} />
            ))}
        </Space>
    );
});
```

Пример блоков с FullWidth.

```jsx live
render(() => {
    const styleBlock = {
        width: '100%',
        height: '32px',
        borderRadius: '8px',
        backgroundColor: 'var(--color-light-specialbg-secondary-transparent)',
    };
    return (
        <Space direction='horizontal' size={12} fullWidth>
            {new Array(5).fill(null).map((_, idx) => (
                <div key={idx} style={styleBlock} />
            ))}
        </Space>
    );
});
```

Пример блоков с выравниванием.

```jsx live
render(() => {
    const styleBlock = {
        width: '96px',
        height: '32px',
        borderRadius: '8px',
        backgroundColor: 'var(--color-light-specialbg-secondary-transparent)',
    };
    return (
        <div style={{ display: 'flex', justifyContent: 'space-between' }}>
            <Space align='start' size={8}>
                <div style={styleBlock} />
                <div style={{ ...styleBlock, width: '144px' }} />
                <div style={{ ...styleBlock, width: '176px' }} />
            </Space>
            <Space align='center' size={8}>
                <div style={styleBlock} />
                <div style={{ ...styleBlock, width: '144px' }} />
                <div style={{ ...styleBlock, width: '176px' }} />
            </Space>
            <Space align='end' size={8}>
                <div style={styleBlock} />
                <div style={{ ...styleBlock, width: '144px' }} />
                <div style={{ ...styleBlock, width: '176px' }} />
            </Space>
        </div>
    );
});
```
