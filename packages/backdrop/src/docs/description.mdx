## Примеры

Компонент поддерживает все параметры из [CSSTransition](http://reactcommunity.org/react-transition-group/css-transition).

```jsx live
render(() => {
    const [open, setOpen] = React.useState(false);
    const isMobile = document.body.clientWidth < 600;

    return (
        <div style={{ zIndex: 1, position: 'relative' }}>
            <Button 
                onClick={() => setOpen(!open)} 
                size='m' 
                block={isMobile} 
                breakpoint={600}
            >
                {open ? 'Закрыть Backdrop' : 'Показать Backdrop'}
            </Button>
            <Backdrop open={open} onClick={() => setOpen(false)} />
        </div>
    );
});
```
