## Tips & Tricks

- Свойство `title` будет использовано для генерации заголовков, а `id` для переключения табов.

- С помощью свойства `disabled` таб можно заблокировать для выбора.

- Контент сегмента (children) будут подставляться под компонент при выборе данного сегмента,
так же можно задать кастомный className для каждого контента сегмента, проьбросив contentClassName
определенному сегменту.
