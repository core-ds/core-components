```jsx
import { TabBar } from '@alfalab/core-components/tab-bar';
```

## Использование dataTestId

В компоненте используется модификатор для `dataTestId`.  
Для удобного поиска элементов можно воспользоваться функцией `getTabBarTestIds`.  
Импорт из `@alfalab/core-components/tab-bar/shared`.

Функция возвращает объект:

```jsx
{
    tabBar: dataTestId, 
    tab: `${dataTestId}-tab`,
};
```
