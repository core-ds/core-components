## Примеры

```jsx live mobileHeight={640}
const options = [{ key: 'Car' }, { key: 'Star' }, { key: 'Flower' }, { key: 'Banknote' }];

const optionsWithIcons = [
    { key: 'Car', icon: CarMIcon },
    { key: 'Star', icon: StarMIcon },
    { key: 'Flower', icon: FlowerMMIcon },
    { key: 'Banknote', icon: BanknoteMIcon },
];

render(
    <Space direction='horizontal'>
        <PickerButtonDesktop options={optionsWithIcons} view='primary' label='Picker button' />
        <PickerButtonDesktop options={options} view='primary' variant='compact' />
    </Space>,
);
//MOBILE
```

## Морфология

Компонент представляет из себя [Select](?path=/docs/select--docs) с [Button](?path=/docs/-button--docs) вместо стандартного `field`.
