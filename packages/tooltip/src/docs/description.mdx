## Виды тултипов

```jsx live
<div style={{display: 'flex'}}>
    <TooltipDesktop
        position='bottom'
        trigger='click'
        content={
            <div style={{ width: 241}}> Пример небольшого вспомогательного текста. </div>
        }
    >
        <Button view='primary'>Показать Tooltip</Button>
    </TooltipDesktop>
    <Gap direction='horizontal' size='m'/>
    <TooltipDesktop content={<div>Пример подсказки</div>} position='bottom' view='hint' trigger='click'>
        <Button view='primary'>Показать Hint</Button>
    </TooltipDesktop>
</div>
//MOBILE
<Container>
    <Tooltip
        position='bottom'
        trigger='click'
        content={
            <div> Пример небольшого вспомогательного текста. </div>
        }
    >
        <div style={{ width: 320 }}>
            <Button block={true} view='primary'>Показать Tooltip</Button>
        </div>
    </Tooltip>
    <Gap size='m'/>
    <Tooltip content={<div>Пример подсказки</div>} position='bottom' view='hint' trigger='click'>
        <div style={{ width: 320 }}>
            <Button block={true} view='primary'>Показать Hint</Button>
        </div>
    </Tooltip>
</Container>
```

## Позиционирование

Если тултип не помещается в переданной позиции `position`, то он поробует отобразится с противоположной стороны.
Можно вручную указать, в каких позициях следует отображать компонент в случае нехватки места, для этого используйте свойство `fallbackPlacements`.

```jsx live
<div
    style={{
        width: '100%',
        height: '300px',
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
    }}
>
    <Tooltip
        position='left'
        fallbackPlacements={['bottom', 'top']}
        trigger='click'
        content={<div style={{ width: 241 }}> Пример небольшого вспомогательного текста. </div>}
    >
        <Button view='primary'>Показать Tooltip</Button>
    </Tooltip>
</div>
```
