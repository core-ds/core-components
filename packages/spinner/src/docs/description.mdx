## Компонент спиннера

Default

```jsx live
<Spinner visible={true} size='s' />
```

Inverted

```jsx live
<div style={{
    backgroundColor: 'var(--color-light-bg-primary-inverted)',
    padding: '18px 20px',
}}>
    <Spinner visible={true} size='s' colors='inverted' />
</div>
```

## Кастомизация

Для указания размера следует задать класс с указанием одинаковых width, height.

Переменные для кастомизации:

-   --spinner-display-visible
-   --spinner-animation-duration
-   --spinner-animation-timing-function
-   --spinner-default-color
-   --spinner-inverted-color

## Примеры

Spinner `size=xs`:

```jsx live
<Spinner size='xs' visible={true} />
```

Spinner `size=s`:

```jsx live
<Spinner size='s' visible={true} />
```

Spinner `size=m`:

```jsx live
<Spinner size='m' visible={true} />
```

Spinner `size=s` in button:

```jsx live
<Button leftAddons={<Spinner visible={true} />} size='s'>
    Документы загружаются...
</Button>
```
