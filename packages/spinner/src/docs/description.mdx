## Примеры

У компонента есть стандартные размеры: 16, 24 и 48px.

```jsx live
const SIZES = [16, 24, 48];

render(
    <Container>
        <Row align='middle'>
            {SIZES.map((size) => (
                <div key={size} style={{ marginRight: 20 }}>
                    <Spinner visible={true} size={size} />
                </div>
            ))}
        </Row>
    </Container>,
);
```

## Использование в других компонентах

Часто используется в [Button](?path=/docs/button--docs).

```jsx live
    <Container>
        <div style={{marginBottom: 20}}>
            <Button view="primary" loading={true} size='m'>
               Label
            </Button>
        </div>
        <div>
            <Button leftAddons={<Spinner visible={true} />} size='m'>
                Дела делаются
            </Button>
        </div>
    </Container>
//MOBILE
    <Container>
        <Button view="primary" block={true} loading={true} size='m'>
            Label
        </Button>
        <br/>
        <Button block={true} leftAddons={<Spinner visible={true} />} size='m'>
            Дела делаются
        </Button>
    </Container>
```
