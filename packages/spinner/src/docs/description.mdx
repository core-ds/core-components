## Примеры

У компонента есть стандартные размеры: 16, 24 и 48px.

```jsx live
const PRESETS = [16, 24, 48];

render(
    <Container>
        <Row align='middle'>
            {PRESETS.map((preset) => (
                <div key={preset} style={{ marginRight: 20 }}>
                    <Spinner.Preset visible={true} preset={preset} />
                </div>
            ))}
        </Row>
    </Container>,
);
```

## Кастомизация

Спиннеру можно задать кастомную высоту, цвет, толщину линии и отступы внутри контейнера.

```jsx live

render(
    <Container>
        <Row align='middle'>
            <div style={{ marginRight: 20 }}>
                <Spinner visible={true} size={28} lineWidth={4} style={{ padding: "var(--gap-2)", color: "var(--color-light-decorative-text-red)" }} />
            </div>
            <div style={{ marginRight: 20 }}>
                <Spinner visible={true} size={24} lineWidth={2} style={{ padding: "var(--gap-4)", color: "var(--color-light-decorative-text-blue)" }} />
            </div>
        </Row>
    </Container>,
);
```

## Использование в других компонентах

Часто используется в [Button](?path=/docs/button--docs).

```jsx live
    <Container>
        <div style={{marginBottom: 20}}>
            <Button view="primary" loading={true} size={56}>
               Label
            </Button>
        </div>
        <div>
            <Button leftAddons={<Spinner.Preset visible={true} />} size={56}>
                Дела делаются
            </Button>
        </div>
    </Container>
//MOBILE
    <Container>
        <Button view="primary" block={true} loading={true} size={56}>
            Label
        </Button>
        <br/>
        <Button block={true} leftAddons={<Spinner.Preset visible={true} />} size={56}>
            Дела делаются
        </Button>
    </Container>
```
