## Подключениe

```jsx
import { Spinner } from '@alfalab/core-components/spinner';
```

или преднастроенный вариант

```jsx
import { SpinnerPreset } from '@alfalab/core-components/spinner/preset';
```

## Использование

```jsx
<Spinner size={20} lineWidth={2} {...restProps} />
```

или

```jsx
<SpinnerPreset preset={48} {...restProps} />
```
