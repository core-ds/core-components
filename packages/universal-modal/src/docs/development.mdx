```jsx
import type { ControlsResponsiveProps, ControlsDesktopProps, ControlsMobileProps } from '@alfalab/core-components/universal-modal';

import { HeaderResponsive, HeaderDesktop } from '@alfalab/core-components/universal-modal';
import type { HeaderResponsiveProps, HeaderPropsDesktop, HeaderPropsMobile } from '@alfalab/core-components/universal-modal';

import { FooterResponsive, FooterDesktop } from '@alfalab/core-components/universal-modal';

```

Из индекса импортируется responsive версия компонента.

## Использование dataTestId

В компоненте используются модификаторы для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getUniversalModalTestIds`.
Импорт из `@alfalab/core-components/universal-modal/shared`.

Функция возвращает объект:

```jsx
{
        modal: dataTestId,
        header: `${dataTestId}-header`,
        title: `${dataTestId}-header-title`,
        content: `${dataTestId}-content`,
        footer: `${dataTestId}-footer`,
};
```
