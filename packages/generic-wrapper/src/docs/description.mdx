## Лейаут

Компонент представляет собой флексовый контейнер с возможностью настройки вертикального и горизонтального расположения дочерних элементов.
Вертикальные и горизонтальные отступы можно настраивать независимо друг от друга.

```jsx live
render(() => {
    const [direction, setDirection] = React.useState('horizontal');

    const addon = (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '8px',
                height: '64px',
                width: '64px',
                boxSizing: 'border-box',
            }}
        />
    );

    return (
        <Container>
            <Row>
                <div style={{ backgroundColor: 'rgba(55, 120, 251, 0.1)', padding: '16px' }}>
                    <GenericWrapper gap={20} column={direction === 'vertical'}>
                        <GenericWrapper>{addon}</GenericWrapper>
                        <GenericWrapper>{addon}</GenericWrapper>
                        <GenericWrapper>{addon}</GenericWrapper>
                    </GenericWrapper>
                </div>
            </Row>
            <div style={{ marginTop: '20px' }}>
                <RadioGroup
                    label='Направление'
                    direction='vertical'
                    name='radioGroupFirst'
                    onChange={(_, { value }) => setDirection(value)}
                    value={direction}
                >
                    <Radio label='Горизонтальное' value='horizontal' />
                    <Radio label='Вертикальное' value='vertical' />
                </RadioGroup>
            </div>
        </Container>
    );
});
```
