## Подключение

```jsx
import { SidePanel } from '@alfalab/core-components/side-panel';
import { SidePanelDesktop } from '@alfalab/core-components/side-panel/desktop';
import { SidePanelMobile } from '@alfalab/core-components/side-panel/mobile';
```

Из индекса импортируется responsive версия компонента.

## Использование dataTestId

В компоненте используются модификаторы для `dataTestId`.  
Для удобного поиска элементов можно воспользоваться функцией `getSidePanelTestIds`.  
Импорт из `@alfalab/core-components/side-panel/shared`.

Функция возвращает объект:

```jsx
{
        modal: dataTestId,
        content: `${dataTestId}-content`,
        footer: `${dataTestId}-footer`,
        controls: `${dataTestId}-controls`,
        header: `${dataTestId}-header`,
        title: `${dataTestId}-header-title`,
        closer: `${dataTestId}-header-closer`,
        backButton: `${dataTestId}-header-back-button`,
};
```
