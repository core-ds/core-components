Компонент может использоваться самостоятельно или в составе более сложных компонентов

## Архитектура

Компонент состоит из слотов. Часто используемые конфигурации слотов могут быть сохранены как пресеты.

## Примеры

### Скругление и обводка

Для плашки можно включить скругление и обводку. Наличие скругления влияет на формат обводки.

```jsx live
<Space fullWidth>
    <Plate
        view='positive'
        title='Компонент плашки'
        border={false}
        subAddons={[<Button>Подробнее</Button>]}
        leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    />
    <Plate
        view='positive'
        title='Компонент плашки'
        subAddons={[<Button>Подробнее</Button>]}
        leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    />
    <Plate
        view='positive'
        title='Компонент плашки'
        borderRadius={false}
        border={false}
        subAddons={[<Button>Подробнее</Button>]}
        leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    />
    <Plate
        view='positive'
        title='Компонент плашки'
        borderRadius={false}
        subAddons={[<Button>Подробнее</Button>]}
        leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    />
</Space>
```

### Фон

Компоненту может быть задан произвольный фон с использованием токенов из набора bg или передано фоновое изображение. При необходимости для плашки может быть включена тень.

```jsx live
<Space fullWidth>
    <Plate
        view='positive'
        hasCloser
        title='Компонент плашки'
        subAddons={[<Button>Подробнее</Button>, <Button>Хорошо</Button>]}
        leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    />

    <Plate
        view='negative'
        hasCloser
        title='Компонент плашки'
        subAddons={[<Button>Хорошо</Button>]}
        leftAddons={<Badge view='icon' iconColor='negative' content={<AlertCircleMIcon />} />}
    />

    <Plate
        view='attention'
        hasCloser
        title='Компонент плашки'
        leftAddons={<Badge view='icon' iconColor='attention' content={<AlertCircleMIcon />} />}
    />
    <Plate
        view='common'
        hasCloser
        title='Компонент плашки'
        leftAddons={
            <Badge view='icon' iconColor='secondary' content={<InformationCircleMIcon />} />
        }
    />
    <Plate
        view='custom'
        hasCloser
        title='Компонент плашки'
        className='custom'
        shadow
        leftAddons={
            <Badge view='icon' iconColor='secondary' content={<InformationCircleMIcon />} />
        }
    />
</Space>
```

### Механики

Классическая

```jsx live
<Plate
    view='positive'
    hasCloser
    title='Компонент плашки'
    leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    buttons={[<Button>Хорошо</Button>, <Button>Подробнее</Button>]}
>
    Самая обыкновенная плашка
</Plate>
```

Collapsed / Expanded

```jsx live
render(() => {
    const [folded1, setFolded1] = React.useState(true);
    const [folded2, setFolded2] = React.useState(false);

    return (
        <Space fullWidth>
            <Plate
                view='negative'
                hasCloser
                title='Компонент плашки'
                foldable
                folded={folded1}
                onToggle={() => setFolded1(!folded1)}
                leftAddons={
                    <Badge view='icon' iconColor='negative' content={<AlertCircleMIcon />} />
                }
                buttons={[<Button>Хорошо</Button>, <Button>Подробнее</Button>]}
            >
                Самая обыкновенная плашка
            </Plate>
            <Plate
                view='negative'
                hasCloser
                title='Компонент плашки'
                foldable
                folded={folded2}
                onToggle={() => setFolded2(!folded2)}
                leftAddons={
                    <Badge view='icon' iconColor='negative' content={<AlertCircleMIcon />} />
                }
                buttons={[<Button>Хорошо</Button>, <Button>Подробнее</Button>]}
            >
                Самая обыкновенная плашка
            </Plate>
        </Space>
    );
});
```
