Компонент может использоваться самостоятельно или в составе более сложных компонентов

## Архитектура

Компонент состоит из слотов. Часто используемые конфигурации слотов могут быть сохранены как пресеты.

## Примеры

### Скругление и обводка

Для плашки можно включить скругление и обводку. Наличие скругления влияет на формат обводки.

```jsx live mobileHeight={510}
render(() => {
    const isMobile = document.body.clientWidth < 450;
    return (
        <Space fullWidth>
            <Plate
                view='positive'
                title='Компонент плашки'
                border={false}
                subAddons={!isMobile && [<Button>Подробнее</Button>]}
                buttons={isMobile && [<Button>Подробнее</Button>]}
                leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
            />
            <Plate
                view='positive'
                title='Компонент плашки'
                subAddons={!isMobile && [<Button>Подробнее</Button>]}
                buttons={isMobile && [<Button>Подробнее</Button>]}
                leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
            />
            <Plate
                view='positive'
                title='Компонент плашки'
                rounded={false}
                border={false}
                subAddons={!isMobile && [<Button>Подробнее</Button>]}
                buttons={isMobile && [<Button>Подробнее</Button>]}
                leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
            />
            <Plate
                view='positive'
                title='Компонент плашки'
                rounded={false}
                subAddons={!isMobile && [<Button>Подробнее</Button>]}
                buttons={isMobile && [<Button>Подробнее</Button>]}
                leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
            />
        </Space>
    );
});
```

### Фон

Компоненту может быть задан произвольный фон с использованием токенов из набора bg или передано фоновое изображение. При необходимости для плашки может быть включена тень.

```jsx live mobileHeight={560}
render(() => {
    const isMobile = document.body.clientWidth < 450;
    return (
        <Space fullWidth>
            <Plate
                view='positive'
                hasCloser
                title='Компонент плашки'
                subAddons={!isMobile && [<Button>Подробнее</Button>, <Button>Хорошо</Button>]}
                buttons={isMobile && [<Button>Подробнее</Button>, <Button>Хорошо</Button>]}
                leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
            />

            <Plate
                view='negative'
                hasCloser
                title='Компонент плашки'
                subAddons={!isMobile && [<Button>Хорошо</Button>]}
                buttons={isMobile && [<Button>Хорошо</Button>]}
                leftAddons={<Badge view='icon' iconColor='negative' content={<AlertCircleMIcon />} />}
            />

            <Plate
                view='attention'
                hasCloser
                title='Компонент плашки'
                leftAddons={<Badge view='icon' iconColor='attention' content={<AlertCircleMIcon />} />}
            />
            <Plate
                view='common'
                hasCloser
                title='Компонент плашки'
                leftAddons={
                    <Badge view='icon' iconColor='secondary' content={<InformationCircleMIcon />} />
                }
            />
            <Plate
                view='custom'
                hasCloser
                title='Компонент плашки'
                leftAddons={
                    <Badge view='icon' iconColor='secondary' content={<InformationCircleMIcon />} />
                }
            />
            <Plate
                view='custom'
                hasCloser
                title='Компонент плашки'
                className='plateGlobal'
                leftAddons={
                    <Badge view='icon' iconColor='secondary' content={<InformationCircleMIcon />} />
                }
            />
        </Space>
    );
});
```

### Механики

Классическая

```jsx live
<Plate
    view='positive'
    hasCloser
    title='Компонент плашки'
    leftAddons={<Badge view='icon' iconColor='positive' content={<CheckmarkOnCircleMIcon />} />}
    buttons={[<Button>Хорошо</Button>, <Button>Подробнее</Button>]}
>
    Самая обыкновенная плашка
</Plate>
```

Collapsed / Expanded

```jsx live
render(() => {
    const [folded, setFolded] = React.useState(true);

    return (
        <Space fullWidth>
            <Plate
                view='negative'
                hasCloser
                title='Компонент плашки'
                foldable
                folded={folded}
                onToggle={() => setFolded(!folded)}
                leftAddons={
                    <Badge view='icon' iconColor='negative' content={<AlertCircleMIcon />} />
                }
                buttons={[<Button>Хорошо</Button>, <Button>Подробнее</Button>]}
            >
                Самая обыкновенная плашка
            </Plate>
        </Space>
    );
});
```
