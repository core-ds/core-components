## Анатомия

Modal построен как Compound Components. Состоит из Header, Content, Footer, содержащих в себе кастомизируемые слоты. Чтобы во время скролла заголовок и кнопки оставались неподвижны, их нужно расположить в `header` и `footer`. Текст и изображения всегда располагаются в `content` части.

```jsx live mobileHeight=640

const Header = () => {
    const { setHasHeader } = React.useContext(ModalContext);

    React.useEffect(() => setHasHeader(true), [setHasHeader]);

    const styleAddons = {
        width: '48px',
        height: '48px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }
    const styleWrapper = {
        display: 'flex',
        justifyContent: 'center',
        margin: 'var(--gap-xs) var(--gap-xs) var(--gap-m)'
    }

    const styleTitle={
        width: '488px',
        borderRadius: '8px',
        backgroundColor: 'rgba(55, 120, 251, 0.1)',
        border: '1px dashed rgb(55, 120, 251)',
        boxSizing: 'border-box',
    }

    return (
        <div style={styleWrapper} >
            <div style={styleAddons} />
            <div style={styleTitle} />
            <div style={styleAddons} />
        </div>
    );
};
```

## Размеры

Десктопный компонент поддерживает размеры S,M,L,XL, FullScreen. В мобильной версии все модальные окна открываются на полный экран.

```jsx live mobileHeight=640

const Text = () => (
    <>
        <Typography.Text tag='p'>
            Сейчас много говорят об отказах банков в проведении операций, блокировках
            интернет-банка. Это связано с тем, что Центральный банк РФ обязывает банки
            выявлять операции своих клиентов, потенциально нарушающие требования
            Федерального закона 115-ФЗ «О противодействии легализации (отмыванию)
            доходов, полученных преступным путем, и финансированию терроризма».
        </Typography.Text>
        <br/>
        <Typography.Text tag='p'>
            Ознакомиться с требованиями ЦБ РФ можно на официальном сайте. Банки обязаны
            предотвращать проведение таких операций и поэтому могут запросить
            дополнительную информацию и документы, ограничить доступ к платежам через
            интернет-банк или даже отказать в исполнении платежа.
        </Typography.Text>
    </>
);

render(() => {
    const SIZES = ['s', 'm', 'l', 'xl', 'fullscreen'];
    const [modal, setModal] = React.useState(false);
    const [sizeModal, setModalSize] = React.useState('s');
    const handleModal = () => setModal(!modal);

    const handleModalSize =(size) => () => { 
        setModalSize(size)
        handleModal()
    }

    return (
        <Container>
            <div style ={{width: '320px'}}>
                {SIZES.map((size) => (
                    <>
                        <Button key={size} block={true} type='button' size='xs' onClick={handleModalSize(size)}>
                            {size !== 'fullscreen'? size.toUpperCase() : 'FullScreen'}
                        </Button>
                        <br/>
                    </>
                ))}
            </div>
            <ModalDesktop open={modal} onClose={handleModal} size={sizeModal}>
                <ModalDesktop.Header hasCloser={true} />
                <ModalDesktop.Content>
                    <Typography.Title tag='div' view='small'>
                        Почему банк проверяет мои операции?
                    </Typography.Title>
                    <br />
                    <Text/>
                </ModalDesktop.Content>
                <ModalDesktop.Footer>
                    <Button view='primary' size='s' onClick={handleModal}>
                        Понятно
                    </Button>
                </ModalDesktop.Footer>
            </ModalDesktop>
        </Container>
    );
});
```
