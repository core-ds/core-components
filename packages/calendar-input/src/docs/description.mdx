## Пример

Пикер используется, когда выбор дат ограничен. Например, если нельзя указывать выходные дни.
При наличии пикера допустимо блокировать ввод с клавиатуры чтобы исключить ввод недоступных значений.

```jsx live mobileHeight={640}
render(() => {
    const [disableUserInput, setDisableUserInput] = React.useState(false);
    const defaultMonth = new Date('December 1, 2022 00:00:00');

    const events = [new Date().setDate(13), new Date().setDate(19)];

    const offDays = [new Date().setDate(8), new Date().setDate(9)];

    const holidays = [
        new Date().setDate(3),
        new Date().setDate(4),
        new Date().setDate(10),
        new Date().setDate(11),
        new Date().setDate(17),
        new Date().setDate(18),
        new Date().setDate(24),
        new Date().setDate(25),
        new Date().setDate(31),
    ];

    return (
        <Space size={24}>
            <CalendarInputDesktop
                label='Укажите дату'
                defaultMonth={defaultMonth}
                events={events}
                offDays={offDays}
                calendarProps={{
                    selectorView: 'month-only',
                    holidays: holidays,
                }}
                disableUserInput={disableUserInput}
            />
            <Switch
                block={true}
                checked={disableUserInput}
                label='Запретить ввод с клавиатуры'
                onChange={() => setDisableUserInput(!disableUserInput)}
            />
        </Space>
    );
});
```

## Календарь

Настройки календаря можно посмотреть на страничке компонента [Calendar](?path=/docs/components-calendar--docs).

## Морфология

Поле ввода является оберткой над [FormControl](?path=/docs/components-formcontrol--docs),
поэтому наследует все доступные для него настройки. Пикер это
[Calendar](?path=/docs/components-calendar--docs) переданный в [Popover](?path=/docs/components-popover--docs).
