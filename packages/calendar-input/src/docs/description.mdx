## Пример

Пикер используется, когда выбор дат ограничен. Например, если нельзя указывать выходные дни. 
При наличии пикера допустимо блокировать ввод с клавиатуры чтобы исключить ввод недоступных значений.

```jsx live mobileHeight=640
render(() => {
    // import startOfDay from 'date-fns/startOfDay';
    const defaultMonth = new Date('December 1, 2022 00:00:00'); 

    const minDate = startOfDay(new Date());
    minDate.setDate(1);
    minDate.setMonth(11);

    const maxDate = startOfDay(new Date());
    maxDate.setDate(31);
    maxDate.setMonth(11);
    
    const events = [new Date().setDate(13), new Date().setDate(19)];

    const offDays = [
        new Date().setDate(8),
        new Date().setDate(9),
    ];

    const holidays = [
        new Date().setDate(3),
        new Date().setDate(4),
        new Date().setDate(10),
        new Date().setDate(11),
        new Date().setDate(17),
        new Date().setDate(18),
        new Date().setDate(24),
        new Date().setDate(25),
        new Date().setDate(31),
    ];

    return (
        <CalendarInputDesktop 
            label='Укажите дату'
            defaultMonth={defaultMonth}  
            events={events}
            offDays={offDays}                 
            minDate={minDate.getTime()}
            maxDate={maxDate.getTime()} 
            calendarProps={{
            selectorView: 'month-only',
            holidays: holidays,
            }}
        />
    );
});
```

## Морфология

Поле ввода является оберткой над [FormControl](/docs/компоненты-инпуты-formcontrol--form-control),
поэтому наследует все доступные для него настройки. Пикер это
[Calendar](/docs/компоненты-календари-calendar--calendar) переданный в [Popover](/docs/компоненты-модалки-и-поповеры-popover--popover).
