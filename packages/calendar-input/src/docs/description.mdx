## Пример

Пикер используется, когда выбор дат ограничен. Например, если нельзя указывать выходные дни.
При наличии пикера допустимо блокировать ввод с клавиатуры чтобы исключить ввод недоступных значений.

```jsx live mobileHeight=640
render(() => {
    const [keyPress, setKeyPress] = React.useState(false);
    const defaultMonth = new Date('December 1, 2022 00:00:00');

    const events = [new Date().setDate(13), new Date().setDate(19)];

    const offDays = [new Date().setDate(8), new Date().setDate(9)];

    const holidays = [
        new Date().setDate(3),
        new Date().setDate(4),
        new Date().setDate(10),
        new Date().setDate(11),
        new Date().setDate(17),
        new Date().setDate(18),
        new Date().setDate(24),
        new Date().setDate(25),
        new Date().setDate(31),
    ];

    const handleChange = (event) => {
        event.preventDefault();
    };

    return (
        <Space size={24}>
            <CalendarInputDesktop
                label='Укажите дату'
                defaultMonth={defaultMonth}
                events={events}
                offDays={offDays}
                onKeyPress={keyPress ? handleChange : null}
                calendarProps={{
                    selectorView: 'month-only',
                    holidays: holidays,
                }}
            />
            <Switch
                block={true}
                checked={keyPress}
                label='Запретить ввод с клавиатуры'
                onChange={() => setKeyPress(!keyPress)}
            />
        </Space>
    );
});
```

## Календарь

Настройки календаря можно посмотреть на страничке компонента [Calendar](/docs/calendar--calendar).

## Морфология

Поле ввода является оберткой над [FormControl](/docs/formcontrol--form-control),
поэтому наследует все доступные для него настройки. Пикер это
[Calendar](/docs/calendar--calendar) переданный в [Popover](/docs/popover--popover).
