## Пример

HOC добавляет в переданный инпут доп. пропсу `suffix?: ReactNode`.

```jsx live
const SuffixInput = withSuffix(Input);

render(
    <div style={{ width: 320 }}>
        <SuffixInput
            block={true}
            placeholder='Введите число'
            label='Количество месяцев'
            labelView={isMobile() ? 'outer' : 'inner'}
            suffix={<Typography.Text view='component-primary' color='secondary'> мес</Typography.Text> }
            size={isMobile() ? 48 : 56}
            breakpoint={BREAKPOINT}/>
    </div>
);
```
