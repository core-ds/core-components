## Размеры

Рекомендуется использовать 48, 40 и 32 размеры. Размеры 72, 64, 56 — не рекомендованы к использованию.

```jsx live
render(() => {
    const SIZES = [48, 40, 32];

    const [checked, setChecked] = React.useState(
        Object.fromEntries(SIZES.map((key) => [key, false])),
    );

    return (
        <Container>
            <Row align='middle'>
                {SIZES.map((size) => (
                    <div key={size} style={{ marginRight: 12 }}>
                        <Tag
                            size={size}
                            checked={checked[size]}
                            onClick={() => setChecked({ ...checked, [size]: !checked[size] })}
                            view='filled'
                            shape='rectangular'
                        >
                            Label
                        </Tag>
                    </div>
                ))}
            </Row>
        </Container>
    );
});
```

## Анатомия

С помощью слотов `leftAddons` и `rightAddons` можно кастомизировать тэг.
Например, добавить иконку. Переданный контент будет отрисован слева или справа от текста тэга.

## Группа фильтров

Тэг может использоваться в группы фильтров. При необходимости, у тэга с одной иконкой можно отрендерить индикатор, и использовать его как кнопку доступа к другим фильтрам. Такой вид доступен только для [Filled] тэгов 32 и 40 размеров.

```jsx live mobileHeight={768}
const PERIOD_OPTIONS = [
    { key: 'week', content: 'Неделя' },
    { key: 'month', content: 'Месяц' },

    writeoff: false,

    period: null,

    let n = 0;
    if (replenishment) n += 1;
    if (writeoff) n += 1;
    n += dynamicKeys.length;
    if (period) n += 1;
    n += accounts.length;

    return n;

        open,
        disabled,
        innerProps,
        valueRenderer,
        selected,
        selectedMultiple,
        placeholder,

            selected,
            selectedMultiple: selectedMultiple || [],

        const filled = Boolean(value);
        const display = filled ? value : placeholder;
        const checked = open || filled;

        return (
            <div style={{ display: 'inline-block' }}>
                <Tag
                    size={size}
                    view='filled'
                    shape='rectangular'
                    checked={checked}
                    disabled={disabled}
                    rightAddons={Arrow}
                    {...innerProps}
                >
                    {display}

        });

    const toggleAppliedDynamic = (id) => {
        setApplied((prev) => ({
            ...prev,
            dynamicKeys: prev.dynamicKeys.filter((k) => k !== id),
        }));
    };

        if (!selected) return null;

        return `Период: ${selected.content}`;

        if (!selectedMultiple.length) return null;

                    view='filled'
                    shape='rectangular'
                    checked={activeCount > 0}
                    indicatorProps={
                        activeCount > 0
                            ? countIndicator
                                ? { mode: 'count', value: activeCount }
                                : { mode: 'dot' }

                    }
                    onClick={openOrTogglePanel}
                />
                {applied.dynamicKeys.map((id) => (

                        key={id}

                        shape='rectangular'
                        checked={true}
                        onClick={() => toggleAppliedDynamic(id)}

                        {DYNAMIC_META[id]}

                ))}
                <Tag
                    size={DESKTOP_TAG_SIZE}
                    view='filled'
                    shape='rectangular'
                    checked={applied.replenishment}
                    onClick={() => setApplied((p) => ({ ...p, replenishment: !p.replenishment }))}

                                setApplied({ ...draft });
                                setPanelOpen(false);
                            }}
                        >
                            Применить
                        </Button>
                        <Button
                            view='secondary'
                            size={56}
                            type='button'
                            onClick={() => setDraft(emptyFilters())}

                            Сбросить

                    </div>
                </SidePanel.Footer>

const PERIOD_OPTIONS = [
    { key: 'week', content: 'Неделя' },
    { key: 'month', content: 'Месяц' },
    { key: 'year', content: 'Год' },
];

const ACCOUNT_OPTIONS = [
    { key: 'acc1', content: 'Счёт 1' },
    { key: 'acc2', content: 'Счёт 2' },
];

const DYNAMIC_META = {
    added_expenses: 'Добавленные расходы',
    to_client: 'Клиенту банка',
    between_accounts: 'Между счетами',
    templates: 'Шаблоны',
    autopay: 'Автоплатежи',
};

const emptyFilters = () => ({
    replenishment: false,
    writeoff: false,
    dynamicKeys: [],
    period: null,
    accounts: [],
});

const countActive = (f) => {
    let n = 0;
    if (f.replenishment) n += 1;
    if (f.writeoff) n += 1;
    n += f.dynamicKeys.length;
    if (f.period) n += 1;
    n += f.accounts.length;
    return n;
};

const MOBILE_ROW_TAG_SIZE = 32;

render(() => {
    const TagFieldSelectArrow = ({ open, size }) => {
        const ChevronIcon = size === 40 ? ChevronDownSIcon : ChevronDownMIcon;

        return (
            <ChevronIcon
                style={{
                    display: 'block',
                    color: 'currentColor',
                    transition: 'transform 0.15s ease-in-out, opacity 0.2s ease',
                    transform: open ? 'rotate(180deg)' : undefined,
                }}
            />
        );
    };

    const TagField = (props) => {
        const {
            open,
            disabled,
            innerProps,
            valueRenderer,
            selected,
            selectedMultiple,
            placeholder,
            Arrow,
            size = MOBILE_ROW_TAG_SIZE,
        } = props;
        const value = valueRenderer({
            selected,
            selectedMultiple: selectedMultiple || [],
        });
        const filled = Boolean(value);
        const display = filled ? value : placeholder;
        const checked = open || filled;

        return (
            <div style={{ display: 'inline-block' }}>
                <TagMobile
                    size={size}
                    view='filled'
                    shape='rectangular'
                    checked={checked}
                    disabled={disabled}
                    rightAddons={Arrow}
                    {...innerProps}
                >
                    {display}
                </TagMobile>
            </div>
        );
    };

    const [panelOpen, setPanelOpen] = React.useState(false);
    const [countIndicator, setCountIndicator] = React.useState(false);
    const [applied, setApplied] = React.useState(emptyFilters);
    const [draft, setDraft] = React.useState(emptyFilters);

    const openOrTogglePanel = () => {
        if (panelOpen) {
            setPanelOpen(false);
        } else {
            setDraft({ ...applied });
            setPanelOpen(true);
        }
    };

    const activeCount = countActive(applied);

    const toggleDraftDynamic = (id) => {
        setDraft((prev) => {
            const has = prev.dynamicKeys.includes(id);
            if (has) {
                return { ...prev, dynamicKeys: prev.dynamicKeys.filter((k) => k !== id) };
            }
            return { ...prev, dynamicKeys: [...prev.dynamicKeys, id] };
        });
    };

    const toggleAppliedDynamic = (id) => {
        setApplied((prev) => ({
            ...prev,
            dynamicKeys: prev.dynamicKeys.filter((k) => k !== id),
        }));
    };

    const periodValueRenderer = ({ selected }) => {
        if (!selected) return null;
        return `Период: ${selected.content}`;
    };

    const accountsValueRenderer = ({ selectedMultiple }) => {
        if (!selectedMultiple.length) return null;
        if (selectedMultiple.length === 1) return `Счета: ${selectedMultiple[0].content}`;
        return `Счета: Выбрано ${selectedMultiple.length}`;
    };

    const selectRowProps = {
        allowUnselect: true,
        block: false,
        size: MOBILE_ROW_TAG_SIZE,
        Option: BaseOption,
        Field: TagField,
        Arrow: TagFieldSelectArrow,
        optionsListWidth: 'field',
    };

    const mobileFilterTagsScrollRowStyle = {
        display: 'flex',
        flexDirection: 'row',
        flexWrap: 'nowrap',
        gap: 8,
        alignItems: 'center',
        overflowX: 'auto',
        overflowY: 'hidden',
        WebkitOverflowScrolling: 'touch',
        touchAction: 'pan-x',
        paddingTop: 10,
        paddingBottom: 4,
        boxSizing: 'border-box',
    };

    const mobileFilterSheetTagsRowStyle = {
        display: 'flex',
        flexDirection: 'row',
        flexWrap: 'wrap',
        gap: 8,
        alignItems: 'center',
    };

    return (
        <Container>
            <div style={mobileFilterTagsScrollRowStyle}>
                <div style={{ display: 'flex' }}>
                    <TagMobile
                        Component={IndicatorTag}
                        leftAddons={<SlidersMIcon height={16} width={16} />}
                        size={MOBILE_ROW_TAG_SIZE}
                        view='filled'
                        shape='rectangular'
                        checked={activeCount > 0}
                        indicatorProps={
                            activeCount > 0
                                ? countIndicator
                                    ? { mode: 'count', value: activeCount }
                                    : { mode: 'dot' }
                                : undefined
                        }
                        onClick={openOrTogglePanel}
                    />
                </div>

                {applied.dynamicKeys.map((id) => (
                    <div key={id}>
                        <TagMobile
                            size={MOBILE_ROW_TAG_SIZE}
                            view='filled'
                            shape='rectangular'
                            checked={true}
                            onClick={() => toggleAppliedDynamic(id)}
                        >
                            {DYNAMIC_META[id]}
                        </TagMobile>
                    </div>
                ))}

                <div>
                    <TagMobile
                        size={MOBILE_ROW_TAG_SIZE}
                        view='filled'
                        shape='rectangular'
                        checked={applied.replenishment}
                        onClick={() =>
                            setApplied((p) => ({ ...p, replenishment: !p.replenishment }))
                        }
                    >
                        Пополнения
                    </TagMobile>
                </div>
                <div>
                    <TagMobile
                        size={MOBILE_ROW_TAG_SIZE}
                        view='filled'
                        shape='rectangular'
                        checked={applied.writeoff}
                        onClick={() => setApplied((p) => ({ ...p, writeoff: !p.writeoff }))}
                    >
                        Списания
                    </TagMobile>
                </div>

                <div>
                    <SelectMobile
                        {...selectRowProps}
                        options={PERIOD_OPTIONS}
                        placeholder='Период'
                        selected={applied.period}
                        valueRenderer={periodValueRenderer}
                        onChange={({ selected: next }) => {
                            setApplied((p) => ({ ...p, period: next ? next.key : null }));
                        }}
                    />
                </div>
                <div>
                    <SelectMobile
                        {...selectRowProps}
                        options={ACCOUNT_OPTIONS}
                        placeholder='Счета'
                        multiple={true}
                        selected={applied.accounts}
                        valueRenderer={accountsValueRenderer}
                        onChange={({ selectedMultiple }) => {
                            setApplied((p) => ({
                                ...p,
                                accounts: selectedMultiple.map((o) => o.key),
                            }));
                        }}
                    />
                </div>
            </div>

            <Gap size={16} />

            <Divider />

            <Gap size={16} />

            <Switch
                checked={countIndicator}
                label='Индикатор с цифрой'
                onChange={() => setCountIndicator((v) => !v)}
            />

            <BottomSheet
                open={panelOpen}
                onClose={() => setPanelOpen(false)}
                title='Фильтры'
                hasCloser={true}
                stickyFooter={true}
                initialHeight='full'
                actionButton={
                    <div
                        style={{
                            display: 'flex',
                            flexDirection: 'row',
                            gap: 16,
                            width: '100%',
                            boxSizing: 'border-box',
                        }}
                    >
                        <ButtonMobile
                            view='secondary'
                            size={48}
                            type='button'
                            block={true}
                            style={{ flex: 1 }}
                            onClick={() => setDraft(emptyFilters())}
                        >
                            Сбросить
                        </ButtonMobile>
                        <ButtonMobile
                            view='primary'
                            size={48}
                            type='button'
                            block={true}
                            style={{ flex: 1 }}
                            onClick={() => {
                                setApplied({ ...draft });
                                setPanelOpen(false);
                            }}
                        >
                            Применить
                        </ButtonMobile>
                    </div>
                }
            >
                <>
                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}
                    >
                        Период
                    </Typography.Text>
                    <Gap size={8} />
                    <div style={mobileFilterSheetTagsRowStyle}>
                        {PERIOD_OPTIONS.map((opt) => (
                            <div key={opt.key}>
                                <TagMobile
                                    size={MOBILE_ROW_TAG_SIZE}
                                    view='filled'
                                    shape='rectangular'
                                    checked={draft.period === opt.key}
                                    onClick={() =>
                                        setDraft((p) => ({
                                            ...p,
                                            period: p.period === opt.key ? null : opt.key,
                                        }))
                                    }
                                >
                                    {opt.content}
                                </TagMobile>
                            </div>
                        ))}
                    </div>

                    <Gap size={24} />

                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}
                    >
                        Движение средств
                    </Typography.Text>
                    <Gap size={8} />
                    <div style={mobileFilterSheetTagsRowStyle}>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.replenishment}
                                onClick={() =>
                                    setDraft((p) => ({ ...p, replenishment: !p.replenishment }))
                                }
                            >
                                Пополнение
                            </TagMobile>
                        </div>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.writeoff}
                                onClick={() => setDraft((p) => ({ ...p, writeoff: !p.writeoff }))}
                            >
                                Списание
                            </TagMobile>
                        </div>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.dynamicKeys.includes('added_expenses')}
                                onClick={() => toggleDraftDynamic('added_expenses')}
                            >
                                Добавленные расходы
                            </TagMobile>
                        </div>
                    </div>

                    <Gap size={24} />

                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}
                    >
                        Переводы
                    </Typography.Text>
                    <Gap size={8} />
                    <div style={mobileFilterSheetTagsRowStyle}>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.dynamicKeys.includes('to_client')}
                                onClick={() => toggleDraftDynamic('to_client')}
                            >
                                Клиенту банка
                            </TagMobile>
                        </div>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.dynamicKeys.includes('between_accounts')}
                                onClick={() => toggleDraftDynamic('between_accounts')}
                            >
                                Между счетами
                            </TagMobile>
                        </div>
                    </div>

                    <Gap size={24} />

                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}
                    >
                        Шаблоны и автоплатежи
                    </Typography.Text>
                    <Gap size={8} />
                    <div style={mobileFilterSheetTagsRowStyle}>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.dynamicKeys.includes('templates')}
                                onClick={() => toggleDraftDynamic('templates')}
                            >
                                Шаблоны
                            </TagMobile>
                        </div>
                        <div>
                            <TagMobile
                                size={MOBILE_ROW_TAG_SIZE}
                                view='filled'
                                shape='rectangular'
                                checked={draft.dynamicKeys.includes('autopay')}
                                onClick={() => toggleDraftDynamic('autopay')}
                            >
                                Автоплатежи
                            </TagMobile>
                        </div>
                    </div>

                    <Gap size={24} />

                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}

                        Счета

                        <div key={opt.key} style={{ marginBottom: 12 }}>
                            <Checkbox
                                checked={draft.accounts.includes(opt.key)}
                                label={opt.content}

                                    setDraft((p) => {

                                        };
                                    })
                                }
                            />
                        </div>
                    ))}

            </BottomSheet>
```
