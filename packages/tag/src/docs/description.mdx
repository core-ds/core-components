## Размеры

Рекомендуется использовать 48, 40 и 32 размеры. Размеры 72, 64, 56 — не рекомендованы к использованию.

```jsx live
render(() => {
    const SIZES = [48, 40, 32];

    const [checked, setChecked] = React.useState(
        Object.fromEntries(SIZES.map((key) => [key, false])),
    );

    return (
        <Container>
            <Row align='middle'>
                {SIZES.map((size) => (
                    <div key={size} style={{ marginRight: 12 }}>
                        <Tag
                            size={size}
                            checked={checked[size]}
                            onClick={() => setChecked({ ...checked, [size]: !checked[size] })}
                            view='filled'
                            shape='rectangular'
                        >
                            Label
                        </Tag>
                    </div>
                ))}
            </Row>
        </Container>
    );
});
```

## Анатомия

С помощью слотов `leftAddons` и `rightAddons` можно кастомизировать тэг.
Например, добавить иконку. Переданный контент будет отрисован слева или справа от текста тэга.

```jsx live
render(() => {
    const [checked, setChecked] = React.useState(false);

    const addon = (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '4px',
                height: '16px',
                width: '16px',
                boxSizing: 'border-box',
            }}
        />
    );

    const defaultProps = {
        size: 40,
        view: 'filled',
        shape: 'rectangular'
    }

    const amountAddon = (
        <div style={{ paddingRight: '4px' }}>
            <Amount view='withZeroMinorPart' value={123400} currency='RUR' minority={100} />
        </div>
    );

    return (
        <Container>
            <Row>
                <div style={{ marginRight: 12 }}>
                    <Tag rightAddons={addon} leftAddons={addon} {...defaultProps}>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag rightAddons={<Badge view='count' height={16} content={1} />} {...defaultProps}>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag rightAddons={<Badge view='count' height={16} />} {...defaultProps}>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag size={40} rightAddons={amountAddon} view='filled' shape='rectangular'>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag
                        Component={IndicatorTag}
                        leftAddons={<ArrowsUpDownMIcon height={16} width={16} />}
                        checked={checked}
                        onClick={(_, { checked: nextChecked }) => setChecked(nextChecked)}
                        {...defaultProps}
                    />
                </div>
            </Row>
        </Container>
    );
});
```

## Состояния и стили

У тега есть две опции, отвечающие за внешний вид.

```jsx live
render(() => {
    const [disabled, setDisabled] = React.useState(false);
    const [shape, setShape] = React.useState('rectangular');
    const [view, setView] = React.useState('filled');
    const [checkedWithAddon, setCheckedWithAddon] = React.useState(false);
    const [checkedPlain, setCheckedPlain] = React.useState(false);

    const tagProps = {
        size: 40,
        shape,
        view,
        disabled,
    };

    return (
        <Container>
            <Row align='middle' style={{ marginBottom: 24 }}>
                <div style={{ marginRight: 12 }}>
                    <Tag
                        {...tagProps}
                        checked={checkedWithAddon}
                        leftAddons={<DiamondsMIcon height={16} width={16} />}
                        onClick={() => setCheckedWithAddon((v) => !v)}
                    >
                        Label
                    </Tag>
                </div>
                <div>
                    <Tag
                        {...tagProps}
                        checked={checkedPlain}
                        onClick={() => setCheckedPlain((v) => !v)}
                    >
                        Label
                    </Tag>
                </div>
            </Row>

            <Gap size={16} />

            <RadioGroup
                direction='horizontal'
                label='Shape'
                name='tag-states-shape'
                onChange={(_, p) => setShape(p.value)}
                value={shape}
            >
                <Radio label='Rectangular' value='rectangular' />
                <Radio label='Rounded' value='rounded' />
            </RadioGroup>

            <Gap size={16} />

            <RadioGroup
                direction='horizontal'
                label='View'
                name='tag-states-view'
                onChange={(_, p) => setView(p.value)}
                value={view}
            >
                <Radio label='Filled' value='filled' />
                <Radio label='Transparent' value='transparent' />
                <Radio label='Outlined' value='outlined' />
            </RadioGroup>

            <Gap size={20} />

            <Switch
                checked={disabled}
                label='Заблокированное состояние'
                onChange={() => setDisabled((v) => !v)}
            />
        </Container>
    );
});
```
