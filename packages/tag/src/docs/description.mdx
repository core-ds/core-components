## Размеры

Рекомендуется использовать 32, 40 и 48 размеры. Размеры 56, 64, 72 — не рекомендованы к использованию.

```jsx live
render(() => {
    const SIZES = [48, 40, 32];

    const [checked, setChecked] = React.useState(
        Object.fromEntries(SIZES.map((key) => [key, false])),
    );

    return (
        <Container>
            <Row align='middle'>
                {SIZES.map((size) => (
                    <div key={size} style={{ marginRight: 12 }}>
                        <TagDesktop
                            size={size}
                            checked={checked[size]}
                            onClick={() => setChecked({ ...checked, [size]: !checked[size] })}
                            view='filled'
                            shape='rectangular'
                        >
                            Label
                        </TagDesktop>
                    </div>
                ))}
            </Row>
        </Container>
    );
});
```

## Механики

Крестик не нужен, если теги используются в качестве чекбоксов или радио-кнопок.

```jsx live
render(() => {
    const [showClear, setShowClear] = React.useState(false);

    return (
```

## Группа фильтров

Тэг может использоваться в группы фильтров. При необходимости, у тэга с одной иконкой можно отрендерить индикатор, и использовать его как кнопку доступа к другим фильтрам. Такой вид доступен только для [Filled] тэгов 32 и 40 размеров.

```jsx live mobileHeight={768}

    { key: 'month', content: 'Месяц' },

    replenishment: false,

    let n = 0;

    if (writeoff) n += 1;
    n += dynamicKeys.length;
    if (period) n += 1;

render(() => {

    const toggleDraftDynamic = (id) => {

                                                onClick={() => {
                                                    if (item.type === 'boolean') {
                                                        updateDraft({
                                                            [item.key]: !draft[item.key],
                                                        });
                                                        return;
                                                    }

                                                    toggleDraftDynamic(item.key);
                                                }}
                                            >
                                                {item.label}
                                            </FilterTagItem>
                                        </div>
                                    );
                                })}
                            </div>
                            <Gap size={24} />
                        </React.Fragment>
                    ))}

                    <Typography.Text
                        tag='p'
                        view='component-secondary'
                        color='secondary'
                        defaultMargins={false}
                    >
                        Счета
                    </Typography.Text>
                    <Gap size={12} />

                    {ACCOUNT_OPTIONS.map((opt) => (
                        <div key={opt.key} style={{ marginBottom: 12 }}>
                            <Checkbox
                                checked={draft.accounts.includes(opt.key)}
                                label={opt.content}
                                onChange={() =>
                                    updateDraft({
                                        accounts: toggleInArray(draft.accounts, opt.key),
                                    })
                                }
                            />
                        </div>
                    ))}
                </>
            </BottomSheet>
        </Container>
    );
});
```
