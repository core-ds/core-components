## Размеры

Рекомендуется использовать S, XS и XXS размеры. Размеры XL, L, M — не рекомендованы к использованию.

```jsx live
render(() => {
    const SIZES = ['s', 'xs', 'xxs'];

    const [checked, setChecked] = React.useState(
        Object.fromEntries(SIZES.map((key) => [key, false])),
    );

    return (
        <Container>
            <Row align='middle'>
                {SIZES.map((size) => (
                    <div key={size} style={{ marginRight: 12 }}>
                        <Tag
                            size={size}
                            checked={checked[size]}
                            onClick={() => setChecked({ ...checked, [size]: !checked[size] })}
                        >
                            Label
                        </Tag>
                    </div>
                ))}
            </Row>
        </Container>
    );
});
```

## Анатомия

С помощью слотов `leftAddons` и `rightAddons` можно кастомизировать тэг.
Например, добавить иконку. Переданный контент будет отрисован слева или справа от текста тэга.

```jsx live
render(() => {
    const addon = (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '8px',
                height: '24px',
                width: '24px',
                boxSizing: 'border-box',
            }}
        />
    );

    const amountAddon = (
        <div style={{ paddingRight: '4px' }}>
            <Amount view='withZeroMinorPart' value={123400} currency='RUR' minority={100} />
        </div>
    );

    return (
        <Container>
            <Row>
                <div style={{ marginRight: 12 }}>
                    <Tag size='s'>Label</Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag size='s' rightAddons={addon} leftAddons={addon}>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag size='s' rightAddons={<Badge view='count' height={24} content={1} />}>
                        Label
                    </Tag>
                </div>
                <div style={{ marginRight: 12 }}>
                    <Tag size='s' rightAddons={amountAddon}>
                        Label
                    </Tag>
                </div>
            </Row>
        </Container>
    );
});
```

## Состояния

Таб может находиться в активном и неактивном состоянии. Взаимодействие может быть ограничено с помощью свойства `disabled`.

```jsx live
render(() => (
    <Container>
        <Row>
            <div style={{ marginRight: 12 }}>
                <Tag size='s'>Label</Tag>
            </div>
            <div>
                <Tag size='s' disabled={true}>
                    Label
                </Tag>
            </div>
        </Row>
        <div style={{ display: 'flex', marginTop: 16 }}>
            <div style={{ marginRight: 12 }}>
                <Tag size='s' checked={true}>
                    Label
                </Tag>
            </div>
            <Col>
                <Tag size='s' checked={true} disabled={true}>
                    Label
                </Tag>
            </Col>
        </div>
    </Container>
));
```

## Стиль

С помощью `variant` можно задать альтернативный стиль тэгу, например, с другим радиусом скругления.

```jsx live
render(() => (
    <Container>
        <Row>
            <div style={{ marginRight: 12 }}>
                <Tag size='s'>Label</Tag>
            </div>
            <div>
                <Tag size='s' variant='alt'>
                    Label
                </Tag>
            </div>
        </Row>
    </Container>
));
```
