import { Container, Row, Col } from 'storybook/blocks';
import { CustomButton } from '@alfalab/core-components-custom-button';

Компонент используется при необходимости отобразить кнопку с кастомным цветом фона.

```jsx live
<Container>
    <Row align='middle'>
        <Col>
            <CustomButton backgroundColor='#FF45C3'>Кнопка</CustomButton>
        </Col>
        <Col>
            <CustomButton backgroundColor='#4AF2FD' contentColor='black'>
                Кнопка
            </CustomButton>
        </Col>
    </Row>
</Container>
```

## Виды кнопок

Размеры и отступы идентичны компоненту [Button](/docs/button--button).

На данный момент поддерживается только `view Primary`.

```jsx live
const SIZES = ['xxs', 'xs', 's', 'm', 'l', 'xl'];

render(
    <Container>
        <Row>
            {SIZES.map((size) => (
                <Col key={size}>
                    <CustomButton size={size}>Кнопка</CustomButton>
                </Col>
            ))}
        </Row>
    </Container>,
);
```

## Цвет фона и контента

-   Кнопка может принимать пропс `backgroundColor` отвечающий за цвет фона.
-   Кнопка может принимать пропс `contentColor = white | black` отвечающий за цвет контента.
-   Кнопка может принимать пропс `stateType = darkening | lightening` отвечающий за затемнение / осветление цвета фона.

```jsx live
const WHITE_CONTENT_COLORS = [
    '#FF8510',
    'linear-gradient(264.15deg, #FF42CA 0%, #FF8A00 100%)',
    '#FF45C3',
];
const BLACK_CONTENT_COLORS = [
    '#4AF2FD',
    'linear-gradient(80.38deg, #4AF2FD 0%, #4AFD9C 100%)',
    '#4AFDA2',
];
const STATE_TYPES = ['darkening', 'lightening'];

render(
    <Container>
        <Row>
            {WHITE_CONTENT_COLORS.map((color) => (
                <Col key={color}>
                    <CustomButton backgroundColor={color}>Кнопка</CustomButton>
                </Col>
            ))}
        </Row>
        <Row>
            {BLACK_CONTENT_COLORS.map((color) => (
                <Col key={color}>
                    <CustomButton backgroundColor={color} contentColor='black'>
                        Кнопка
                    </CustomButton>
                </Col>
            ))}
        </Row>
        <Row>
            {STATE_TYPES.map((stateType) => (
                <Col key={stateType}>
                    <CustomButton backgroundColor='#FF45C3' stateType={stateType} contentColor='white'>
                        Кнопка
                    </CustomButton>
                </Col>
            ))}
        </Row>
    </Container>,
);
```
