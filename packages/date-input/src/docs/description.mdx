## Стандартный вид компонента

Поле ввода с маской ДД.ММ.ГГГГ.

```jsx live
<div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
    <DateInput label='Дата зимнего солнцестояния' size='m' />
</div>
```

## Морфология

Поле ввода является оберткой над [FormControl](?path=/docs/%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D1%8B-formcontrol--form-control),
поэтому наследует все доступные для него настройки. Пикер это
[Calendar](?path=/docs/%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D1%8B-calendar--calendar)
в [Popover](?path=/docs/%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D1%8B-popover--popover).
