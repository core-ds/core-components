## Стандартный вид компонента

Поле ввода с маской ДД.ММ.ГГГГ.

```jsx live
<div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
    <DateInput label='Дата зимнего солнцестояния' size='m' />
</div>
```

## Морфология

Поле ввода является оберткой над [FormControl](?path=/docs/components-formcontrol--docs),
поэтому наследует все доступные для него настройки. Пикер это
[Calendar](?path=/docs/components-calendar--docs)
в [Popover](?path=/docs/components-popover--docs).
