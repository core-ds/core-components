# Приведённый код и live-демо имеют демонстрационный характер и не являются финальной реализацией.

## Shake

Shake-анимация с эффектом «выброса» — кнопка смещается по оси X и плавно возвращается благодаря spring-физике.

```jsx live mobileHeight={500}
render(() => {
    const [state, setState] = React.useState({ stiffness: 100, damping: 10, mass: 1 });
    const [playing, setPlaying] = React.useState(false);
    const cancelRef = React.useRef(null);

    return (
        <>
            <div style={{ padding: '16px' }}>
                <div
                    style={{
                        display: 'flex',
                        alignItems: 'center',
                        gap: '12px',
                        flexWrap: 'wrap',
                        marginBottom: '16px',
                    }}
                >
                    <ButtonDesktop
                        view='primary'
                        shake={true}
                        shakeSpring={state}
                        onSpringAnimationStart={(cancel) => {
                            setPlaying(true);
                            cancelRef.current = cancel;
                        }}
                        onSpringAnimationEnd={() => setPlaying(false)}
                    >
                        Click me!
                    </ButtonDesktop>
                    {playing && (
                        <>
                            <span style={{ fontSize: '13px', color: '#888' }}>
                                Анимация воспроизводится...
                            </span>
                            <ButtonDesktop
                                view='outlined'
                                size={40}
                                onClick={() => cancelRef.current && cancelRef.current()}
                            >
                                Отменить
                            </ButtonDesktop>
                        </>
                    )}
                </div>
                <div style={{ display: 'flex', gap: '24px', flexWrap: 'wrap' }}>
                    <div style={{ minWidth: '180px' }}>
                        <div style={{ fontSize: '12px', marginBottom: '4px' }}>
                            Stiffness: {state.stiffness}
                        </div>
                        <Gap size={8} />
                        <Slider
                            value={state.stiffness}
                            min={1}
                            max={300}
                            step={1}
                            onChange={({ value }) => setState((s) => ({ ...s, stiffness: value }))}
                        />
                    </div>
                    <div style={{ minWidth: '180px' }}>
                        <div style={{ fontSize: '12px', marginBottom: '4px' }}>
                            Damping: {state.damping}
                        </div>
                        <Gap size={8} />
                        <Slider
                            value={state.damping}
                            min={1}
                            max={50}
                            step={1}
                            onChange={({ value }) => setState((s) => ({ ...s, damping: value }))}
                        />
                    </div>
                    <div style={{ minWidth: '180px' }}>
                        <div style={{ fontSize: '12px', marginBottom: '4px' }}>
                            Mass: {state.mass.toFixed(1)}
                        </div>
                        <Gap size={8} />
                        <Slider
                            value={state.mass}
                            min={0.1}
                            max={5}
                            step={0.1}
                            onChange={({ value }) => setState((s) => ({ ...s, mass: value }))}
                        />
                    </div>
                </div>
            </div>
        </>
    );
});
```

                            value={state.stiffness}
                            min={1}
                            max={400}
                            step={1}

                        />
                    </div>

        </>

## Модуль анимаций: `useSpringAnimation`

Хук реализован в `packages/button/src/hooks/useSpringAnimation.ts` и инкапсулирует всю логику spring-анимаций на основе библиотеки [motion](https://motion.dev/docs/animate).

### Зачем хук, а не инлайн-код

Каждая анимация — это цепочка вызовов `animate()` (motion не поддерживает больше двух keyframes в spring-режиме). Хук берёт на себя:

- построение цепочки шагов из пресета
- блокировку повторного запуска пока анимация идёт
- отмену с мгновенным сбросом трансформов
- cleanup при размонтировании компонента

### API хука

```tsx
const { trigger, cancel, isPlaying } = useSpringAnimation(
    ref, // RefObject<HTMLElement> — элемент для анимации
    type, // 'shake' | 'pulse' | 'bounce' | 'wobble'
    springOptions, // { stiffness?, damping?, mass? } — переопределяют дефолты пресета
    callbacks, // { onStart?(cancel): void, onEnd?(): void }
);
```

### Использование в компоненте

```tsx
import { useRef } from 'react';
import { useSpringAnimation } from './hooks';

function MyComponent() {
    const ref = useRef<HTMLDivElement>(null);

    const { trigger, cancel, isPlaying } = useSpringAnimation(
        ref,
        'shake',
        { stiffness: 150, damping: 12 },
        {
            onStart: (cancelFn) => console.log('started', cancelFn),
            onEnd: () => console.log('finished'),
        },
    );

    return (
        <div ref={ref}>
            <button onClick={trigger}>Shake</button>
            {isPlaying && <button onClick={cancel}>Отменить</button>}
        </div>
    );
}
```

### Добавление нового пресета

Чтобы добавить новый тип анимации, достаточно расширить `PRESETS` в файле хука и добавить тип в `AnimationType`:

```ts
// packages/button/src/hooks/useSpringAnimation.ts

export type AnimationType = 'shake' | 'pulse' | 'bounce' | 'wobble' | 'flip';

const PRESETS: Record<AnimationType, AnimationPreset> = {
    // ...существующие пресеты...
    flip: {
        defaultSpring: { stiffness: 200, damping: 20, mass: 1 },
        steps: [
            { property: 'rotateY', from: 0, to: 90 },
            { property: 'rotateY', from: -90, to: 0 },
        ],
    },
};
```

Новый тип сразу доступен в хуке и может использоваться в любом компоненте.
