## Типы отображения точек

Компонент поддерживает два типа отображения точек:

-   **`'step'`** - точки отображаются по шагу слайдера (по умолчанию)
-   **`'custom'`** - произвольное расположение точек с помощью параметра `customDots`

При использовании `dotsSlider='custom'` необходимо передать массив значений в `customDots` для указания позиций точек.
