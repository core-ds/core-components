## Сегменты и шаг слайдера

Можно настроить количество сегментов и шаг внутри сегмента.

```jsx live
render(() => {
    const [value, setValue] = React.useState(0);
    const [slider, setSlider] = React.useState(1);

    const handleChange = ({ value }) => setValue(value);
    const handleChangeSlider = ({ value }) => setSlider(value);

    return (
        <>
            <div>Value:{value}</div>
            <br />
            <Slider
                size={4}
                value={value}
                onChange={handleChange}
                pips={{
                    mode: 'values',
                    values: [0, 10, 100],
                }}
                range={{
                    min: [0],
                    '50%': [10, 10],
                    max: [100],
                }}
            />
            <br />
            <br />
            <div>Value:{slider}</div>
            <br />
            <Slider
                size={4}
                value={slider}
                onChange={handleChangeSlider}
                min={1}
                max={8}
                pips={{
                    mode: 'steps',
                }}
            />
        </>
    );
});
```

Можно настроить слайдер с неравномерными шагами.

```jsx live
render(() => {
    const [value, setValue] = React.useState(0);

    const handleChange = ({ value }) => setValue(value);

    return (
        <>
            <div>Value:{value}</div>
            <br />
            <Slider
                size={4}
                value={value}
                onChange={handleChange}
                range={{ min: 0, '10%': 1, '40%': 4, '50%': 5, max: 10 }}
                snap={true}
                pips={{
                    mode: 'values',
                    values: [0, 1, 4, 5, 10],
                }}
            />
        </>
    );
});
```

## Диапазон

С помощью слайдера можно указывать диапазон значений. Чтобы позволить ползункам пересекать друг друга установите `behaviour=unconstrained-tap`.

```jsx live
render(() => {
    const [sliderTap, setSliderTap] = React.useState({
        value: 4,
        valueTo: 6,
    });

    const [slider, setSlider] = React.useState({
        value: 4,
        valueTo: 6,
    });

    const handleChangeSliderTap = (payload) => {
        setSliderTap(payload);
    };

    const handleChangeSlider = (payload) => {
        setSlider(payload);
    };

    return (
        <>
            <div style={{ display: 'flex', justifyContent: 'space-between' }}>
                <span>min: {sliderTap.value}</span>
                <span>max: {sliderTap.valueTo}</span>
            </div>
            <Gap size='s' />
            <Slider
                size={4}
                value={sliderTap.value}
                valueTo={sliderTap.valueTo}
                onChange={handleChangeSliderTap}
                min={1}
                max={8}
                pips={{
                    mode: 'steps',
                }}
            />
            <Gap size='3xl' />
            <div style={{ display: 'flex', justifyContent: 'space-between' }}>
                <span>min: {slider.value}</span>
                <span>max: {slider.valueTo}</span>
            </div>
            <Gap size='s' />
            <Slider
                size={4}
                value={slider.value}
                valueTo={slider.valueTo}
                behaviour='unconstrained-tap'
                onChange={handleChangeSlider}
                min={1}
                max={8}
                pips={{
                    mode: 'steps',
                }}
            />
        </>
    );
});
```

### Кастомные точки и управление отображением

#### Типы отображения точек

Слайдер поддерживает два типа отображения точек:

-   **`'step'`** - точки отображаются по шагу слайдера (по умолчанию)
-   **`'custom'`** - произвольное расположение точек

#### Произвольные точки

Можно задать произвольное расположение точек с помощью `dotsSlider='custom'` и `customDots`. В этом случае отображаются только точки в указанных позициях без чисел.

```jsx live
render(() => {
    const [value, setValue] = React.useState(4);

    const handleChange = ({ value }) => setValue(value);

    return (
        <>
            <div>Value: {value}</div>
            <br />
            <Slider
                min={1}
                max={8}
                step={0.5}
                value={value}
                onChange={handleChange}
                size={4}
                dots
                dotsSlider='custom'
                showNumbers={true}
                customDots={[4, 5.5, 7]}
                pips={{ mode: 'values', values: [1, 2, 3, 5, 6] }}
            />
        </>
    );
});
```
