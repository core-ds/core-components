## Сегменты и шаг слайдера

Можно настроить количество сегментов и шаг внутри сегмента.

```jsx live
render(() => {
    const [value, setValue] = React.useState(0);
    const [slider, setSlider] = React.useState(1);

    const handleChange = ({ value }) => setValue(value);
    const handleChangeSlider = ({ value }) => setSlider(value);

    return (
        <>
            <div>Value:{value}</div>
            <br/>
            <Slider
                size='m'
                value={value}
                onChange={handleChange}
                pips={{
                    mode: 'values',
                    values: [0, 10, 100],
                }}
                range={{
                    min: [0],
                    '50%': [10, 10],
                    max: [100],
                }}
            />
            <br/>
            <br/>
            <div>Value:{slider}</div>
            <br/>
            <Slider
                size='m'
                value={slider}
                onChange={handleChangeSlider}
                min={1}
                max={8}
                pips={{
                    mode: 'steps',
                }}
            />
        </>
    );
});
```

## Подписи

Слайдер может использоваться с подписями и без.

```jsx live
render(() => {
    const [value, setValue] = React.useState(0);
    const [slider, setSlider] = React.useState(0);

    const handleChange = ({ value }) => setValue(value);
    const handleChangeSlider = ({ value }) => setSlider(value);

    return (
        <>
            <div>Value: {value}</div>
            <br />
            <Slider
                size='m'
                value={value}
                onChange={handleChange}
                pips={{
                    mode: 'values',
                    values: [0, 10, 100],
                }}
                range={{
                    min: [0],
                    '50%': [10, 10],
                    max: [100],
                }}
            />
            <br />
            <br />
            <div>Value: {slider}</div>
            <br />
            <Slider size='m' value={slider} onChange={handleChangeSlider} />            
        </>
    );
});
```

## Размеры

Доступны M и S размеры компонента.

```jsx live
render(() => {
    const [value, setValue] = React.useState(0);
    const [slider, setSlider] = React.useState(0);

    const handleChange = ({ value }) => setValue(value);
    const handleChangeSlider = ({ value }) => setSlider(value);

    return (
        <>
            <Slider
                size='m'
                value={value}
                onChange={handleChange}
            />
            <br />
            <br />
            <Slider
                size='s'
                value={slider}
                onChange={handleChangeSlider}
            />
        </>
    );
});
//MOBILE
render(() => {
    const [value, setValue] = React.useState(0);
    const [slider, setSlider] = React.useState(0);

    const handleChange = ({ value }) => setValue(value);
    const handleChangeSlider = ({ value }) => setSlider(value);

    return (
        <>
            <div style={{marginBottom: 80}}>
                <Slider
                    size='m'
                    value={value}
                    onChange={handleChange}
                />
            </div>
            <Slider
                size='s'
                value={slider}
                onChange={handleChangeSlider}
            />
        </>
    );
});
```
