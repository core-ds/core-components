## Размеры

Рекомендуется использовать 32, 40 и 48 размеры. Размеры 56, 64, 72 — не рекомендованы к использованию.

```jsx live
const CarouselItem = ({ children, bgColor, itemColor }) => {
    return (
        <div
            style={{
                background: bgColor,
                padding: 'var(--gap-16)',

                borderRadius: 'var(--border-radius-20)',
                height: 'calc(100% - var(--gap-16) * 2)',

                label='Количество видимых элементов'
                direction='vertical'
                name='height'
                onChange={handleVisibleItemsChange}
                value={visibleItems}

                <Radio size={24} label='Фиксированное' value='fixed' />
```

## Удаление контента

Элементами карусели могут быть карточки счетов на оплату, подписки или другие сущности, которые пользователь должен иметь возможность удалить.

```jsx live
const CarouselItem = ({ children, bgColor, itemColor }) => {
    return (
        <div
            style={{
                background: bgColor,
                padding: 'var(--gap-16)',
                display: 'flex',
                flexDirection: 'column',
                borderRadius: 'var(--border-radius-20)',
                height: 'calc(100% - var(--gap-16) * 2)',
                position: 'relative',
            }}
        >
            <SuperEllipse backgroundColor={itemColor} />
            <div
                style={{
                    height: 12,
                    background: itemColor,
                    margin: 'auto 0 var(--gap-6)',
                    borderRadius: 'var(--border-radius-pill)',
                }}
            />
            <div
                style={{
                    height: 12,
                    background: itemColor,
                    margin: 'var(--gap-6) 0',
                    borderRadius: 'var(--border-radius-pill)',
                }}
            />
            {children}
        </div>
    );
};

render(() => {
    const [slides, setSlides] = React.useState(() =>
        [
            {
                bgColor: 'var(--color-light-decorative-muted-blue)',
                itemColor: 'var(--color-light-decorative-muted-alt-blue)',
                height: 200,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-red)',
                itemColor: 'var(--color-light-decorative-muted-alt-red)',
                height: 280,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-orange)',
                itemColor: 'var(--color-light-decorative-muted-alt-orange)',
                height: 160,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-green)',
                itemColor: 'var(--color-light-decorative-muted-alt-green)',
                height: 240,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-indigo)',
                itemColor: 'var(--color-light-decorative-muted-alt-indigo)',
                height: 200,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-cyan)',
                itemColor: 'var(--color-light-decorative-muted-alt-cyan)',
                height: 160,
            },
            {
                bgColor: 'var(--color-light-decorative-muted-pistachio)',
                itemColor: 'var(--color-light-decorative-muted-alt-pistachio)',
                height: 240,
            },
        ].map((slide, index) => ({ ...slide, key: index, order: index })),
    );

    const handleRemove = (key) => {
        const slide = slides.find((slide) => slide.key === key);

        setSlides(slides.filter((slide) => slide.key !== key));

        setTimeout(() => {
            setSlides((prevSlides) => [...prevSlides, slide].sort((a, b) => a.order - b.order));
        }, 5000);
    };
    return (
        <Carousel
            items={slides.map(({ key, ...itemProps }) => ({
                key,
                height: itemProps.height,
                width: 256,
                children: (
                    <CarouselItem {...itemProps}>
                        {itemProps.order < 2 && (
                            <div
                                role='button'
                                title='Удалить'
                                onClick={() => {
                                    handleRemove(key);
                                }}
                                style={{
                                    position: 'absolute',
                                    top: '50%',
                                    left: '50%',
                                    translate: '-50% -50%',
                                    background: 'var(--color-light-neutral-translucent-100)',
                                    borderRadius: '50%',
                                    width: 32,
                                    height: 32,
                                    display: 'flex',
                                    justifyContent: 'center',
                                    alignItems: 'center',
                                    zIndex: 1,
                                }}
                            >
                                <CrossMIcon />
                            </div>
                        )}
                    </CarouselItem>
                ),
            }))}
            visibleItems={'auto'}
            gap={8}
            height={200}
        />
    );
});
```
