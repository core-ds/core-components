## Использование dataTestId

В компоненте используются модификаторы для `dataTestId`.
Для удобного поиска элементов можно воспользоваться функцией `getPassCodeTestIds`.
Импорт из `@balafla/core-components/pass-code/shared`.

Функция возвращает объект:

```jsx
{
    passCode: `${dataTestId}-wrapper`,
    inputProgress: `${dataTestId}-input-progress`,
    keypad: `${dataTestId}-keypad`,
    keypadButton: `${dataTestId}-keypad-button`,
    backspaceButton: `${dataTestId}-backspace-button`,
    error: `${dataTestId}-error`,
    message: `${dataTestId}-message`,
};
```
