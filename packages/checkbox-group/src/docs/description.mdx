## Как использовать

Группа чекбоксов используется когда пользователю доступно несколько вариантов выбора.
Обычно от двух до пяти. Если вариантов больше, возможно следует использовать другой контрол,
например [Select](?path=/docs/components-selects-select--select).

Для выбора пункта необходимо кликнуть по названию пункта или по чекбоксу. Повторное нажатие снимет выбор.

Группа может содержать заголовок и подпись.

В группе может быть выбрано несколько пунктов.

## Горизонтальная группа

Горизонтальное расположение используется при коротких названиях пунктов и малом количестве вариантов — от двух до трёх.

```jsx live
render(() => {
    const [value, setValue] = React.useState({ one: false, two: false, three: false });
    const [error, setError] = React.useState(false);

    const onChange = (_, payload) => {
        setValue({ ...value, [payload.name]: payload.checked });
    };

    return (
        <div style={{ width: '100%' }}>
            <CheckboxGroup
                label='Планеты расположенные между Землёй и Солнцем'
                direction='horizontal'
                onChange={onChange}
                hint='Выберите несколько планет'
                error={error}
            >
                <Checkbox label='Меркурий' name='one' checked={value.one} size='m' />
                <Checkbox label='Венера' name='two' checked={value.two} size='m' />
                <Checkbox label='Нептун' name='three' checked={value.three} size='m' />
            </CheckboxGroup>

            <Gap size='2xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' onClick={() => setError(false)}>
                    По умолчанию
                </Button>
                <Button size='xxs' onClick={() => setError('Правильный ответ: Венера и Меркурий')}>
                    Состояние ошибки
                </Button>
            </Space>
        </div>
    );
});
```

## Вертикальная группа

Вертикальное расположение используется при длинных названиях пунктов. Рекомендуемое количество вариантов от двух до пяти.

```jsx live
render(() => {
    const [value, setValue] = React.useState({
        one: false,
        two: false,
        three: false,
        four: false,
        five: false,
    });
    const [error, setError] = React.useState(false);

    const onChange = (_, payload) => {
        setValue({ ...value, [payload.name]: payload.checked });
    };

    return (
        <div style={{ width: '100%' }}>
            <CheckboxGroup
                label='Выберите планеты земной группы'
                onChange={onChange}
                hint='Несколько из этих планет принадлежат земной группе'
                error={error}
            >
                <Checkbox label='Марс' name='one' checked={value.one} size='m' />
                <Checkbox label='Юпитер' name='two' checked={value.two} size='m' />
                <Checkbox label='Меркурий' name='three' checked={value.three} size='m' />
                <Checkbox label='Сатурн' name='four' checked={value.four} size='m' />
                <Checkbox label='Нептун' name='five' checked={value.five} size='m' />
            </CheckboxGroup>

            <Gap size='2xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' onClick={() => setError(false)}>
                    По умолчанию
                </Button>
                <Button size='xxs' onClick={() => setError('Правильный ответ: Марс и Меркурий')}>
                    Состояние ошибки
                </Button>
            </Space>
        </div>
    );
});
```

## Группа тегов

Теги могут использоваться в качестве радиокнопок в составе группы.
Группа тегов используется при коротких названиях пунктов и малом количестве вариантов — от двух до трех.
Если варианты не помещаются в одну строку они могут перенестись на следующую.

```jsx live
render(() => {
    const [value, setValue] = React.useState({ one: false, two: false, three: false, four: false });
    const [error, setError] = React.useState(false);

    const onChange = (_, payload) => {
        setValue({ ...value, [payload.name]: payload.checked });
    };

    return (
        <div style={{ width: '100%' }}>
            <CheckboxGroup
                label='Выберите один или несколько вариантов'
                onChange={onChange}
                direction='horizontal'
                type='tag'
                hint='Выберите один из вариантов'
                error={error}
            >
                <Tag name='one' checked={value.one}>
                    Вариант 1
                </Tag>
                <Tag name='two' checked={value.two}>
                    Вариант 2
                </Tag>
                <Tag name='three' checked={value.three}>
                    Вариант 3
                </Tag>
                <Tag name='four' checked={value.four}>
                    Вариант 4
                </Tag>
            </CheckboxGroup>

            <Gap size='2xl' />

            <Space direction='horizontal' wrap={true}>
                <Button size='xxs' onClick={() => setError(false)}>
                    По умолчанию
                </Button>
                <Button size='xxs' onClick={() => setError('Error message')}>
                    Состояние ошибки
                </Button>
            </Space>
        </div>
    );
});
```

## Использование в формах

Не рекомендуется использовать в одной форме несколько вариантов реализации радиогрупп.
Чтобы пользователю было комфортно взаимодействовать с интерфейсом, контролы с одинаковой механикой должны выглядеть одинаково.
