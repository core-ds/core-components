Компонент текстового ввода диапазона дат

## Примеры

### Без пикера
```jsx live
render(
    <div style={{ width: 250 }}>
        <DateRangeInput label="Укажите диапазон дат" placeholder="ДД.ММ.ГГГГ - ДД.ММ.ГГГГ" block />
    </div>
)
```


### С пикером
```jsx live
render(
    <div style={{ width: 275 }}>
        <DateRangeInput label="Укажите диапазон дат" placeholder="ДД.ММ.ГГГГ - ДД.ММ.ГГГГ" picker block />
    </div>
)
```


### Пикер
Для настроек пикера см. компонент DatePicker
