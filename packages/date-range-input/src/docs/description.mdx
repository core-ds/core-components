## Стандартный вид компонента

Поле ввода с маской ДД.ММ.ГГГГ – ДД.ММ.ГГГГ. Использование пикера опционально.

```jsx live
render(
    <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
        <DateRangeInput
            label='Период действия патента'
            placeholder='ДД.ММ.ГГГГ - ДД.ММ.ГГГГ'
            block={true}
            size='m'
        />
    </div>,
);
```

## Пикер

Пикер используется, когда выбор дат ограничен. Например, если нельзя указывать выходные дни.
При наличии пикера допустимо блокировать ввод с клавиатуры.

```jsx live mobileHeight={640}
render(() => {
    const [disableUserInput, setDisableUserInput] = React.useState(false);

    return (
        <div style={{ width: '100%' }}>
            <div style={{ width: document.body.clientWidth < 450 ? '100%' : 320 }}>
                <DateRangeInput
                    label='Укажите диапазон дат'
                    placeholder='ДД.ММ.ГГГГ - ДД.ММ.ГГГГ'
                    picker={true}
                    block={true}
                    size='m'
                    disableUserInput={disableUserInput}
                />
            </div>
            <Gap size='xl' />

            <Switch
                checked={disableUserInput}
                onChange={() => setDisableUserInput((prevState) => !prevState)}
                label='Запретить ввод с клавиатуры'
            />
        </div>
    );
});
```
