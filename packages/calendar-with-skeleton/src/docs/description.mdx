## Примеры

```jsx live desktopOnly
render(() => {
    const [visible, setVisible] = React.useState(true);

    return (
        <div style={{ width: 344 }}>
            <CalendarWithSkeleton calendarVisible={visible} />
            <Gap size='xl' />
            <Switch
                block={true}
                checked={!visible}
                label='Скелетонизация контента'
                onChange={() => setVisible(!visible)}
            />
        </div>
    );
});
```
