Календарь с возможностью скелетной загрузки

```jsx live
render(() => {
    const [visible, setVisible] = React.useState(false);
    return (
        <>
            <CalendarWithSkeleton calendarVisible={visible} />
            <Gap size='l'/>
            <Button size='xs' onClick={() => setVisible(!visible)}>
                toggle
            </Button>
        </>
    );
});
```

## Использование с другими компонентами

```jsx live
const CalendarCustom = React.forwardRef((props, ref) => {
    return (
        <CalendarWithSkeleton {...props} ref={ref} />
    );
});

render(() => {
    return <CalendarInput Calendar={CalendarCustom} calendarProps={{ calendarVisible: false }} />;
});
```

```jsx live
<CalendarRange
    inputFromProps={{
        Calendar: CalendarWithSkeleton,
        disabled: true,
        calendarProps: { calendarVisible: false },
    }}
/>
```
