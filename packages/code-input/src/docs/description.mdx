### CodeInput

```tsx live
<CodeInput />
```

Компонент поддерживает автоподстановку кода из СМС в браузерах с поддержкой WebOTP Api.

Пример сообщения
___
Your OTP is: 1234.

@example.com #1234
___
Требования к формату сообщения
- Начинается с удобочитаемого текста, для того чтобы пользователь понял о чем речь;
- Содержит состоящий из 4–10 цифр код OTP;
- Последняя строка соответсвует формату: @`доменная часть URL-адреса веб-сайта, вызвавшего API` #`OTP`.

Более подробно об WebOTP Api можно почитать [тут](https://web.dev/i18n/ru/web-otp/).
