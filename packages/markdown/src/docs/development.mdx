## Подключение

```jsx
import { Markdown } from '@balafla/core-components/markdown';
import { MarkdownMobile } from '@balafla/core-components/markdown/mobile';
import { MarkdownDesktop } from '@balafla/core-components/markdown/desktop';
```

## Стилизация

Для стилизации определенного тега используйте класс с его названием.

> Например для тега h1 -> `.h1 {}`

## Переопределение

Для каждого тега можно переопределить компонент.

В `overrides` передать

```
{ h1: (props) => (<div>{props.children}</div>)}
```
