## Пример

Все интерактивные элементы должны иметь определенную обводку при фокусе, полученном при переходе к элементу при помощи клавиатуры. 
При клике мышкой или тапе обводки быть не должно. 
В примере ниже кликните на первую кнопку мышкой, а затем нажмите tab, чтобы выбрать правую кнопку.

```jsx live
<KeyboardFocusable>
    {(ref, focused) => {
        const focusOutlineStyles = {
            outline: '2px solid var(--focus-color)',
            outlineOffset: '2px',
        };

        return (
            <Space size={16} direction='horizontal'>
                <Button size='m'>
                    Кликни мышкой
                </Button>
                <Button size='m' ref={ref} style={focused ? focusOutlineStyles : {}}>
                    Выбери табом
                </Button>
            </Space>
            
        );
    }}
</KeyboardFocusable>
```
