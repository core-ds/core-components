Набор компонентов для оформления текста.

Для интерактивных элементов, требующих изменение цвета при наведении, следует устанавливать цвет через CSS. Пропса `color` — для статичных элементов.

> **_NOTE:_** Компоненты не устанавливают свойство `font-family`, кроме случаев, когда используется шрифт 'Styrene UI'. Подразумевается, что шрифт будет установлен у родителя.

[Подробнее про использование типографики](https://core-ds.github.io/core-components/master/?path=/docs/инструкции-типографика--page)

## Title

### Styrene

```jsx live
render(() => {
    const VIEW_TYPES = ['xlarge', 'large', 'medium', 'small', 'xsmall'];

    return VIEW_TYPES.map((view) => (
        <Typography.Title view={view} color={color} key={view} tag='div'>
            {view}
        </Typography.Title>
    ));
});
```

### System

```jsx live
render(() => {
    const VIEW_TYPES = ['xlarge', 'large', 'medium', 'small', 'xsmall'];

    return VIEW_TYPES.map((view) => (
        <Typography.Title font='system' view={view} color={color} key={view} tag='div'>
            {view}
        </Typography.Title>
    ));
});
```

## Text

### System

Компонент для оформления текста. Позволяет задать внешний вид, тэг и цвет текста.

```jsx live
render(() => {
    const VIEW_TYPES = [
        'primary-large',
        'primary-medium',
        'primary-small',
        'secondary-large',
        'secondary-medium',
        'secondary-small',
        'component',
        'caps',
    ];
    return VIEW_TYPES.map((view) => (
        <Typography.Text view={view} key={view} tag='p'>
            {view}. Космологи́ческая сингуля́рность — состояния Вселенной в определённый момент
            времени в прошлом, когда плотность энергии (материи) и кривизна пространства-времени
            были очень велики — порядка планковских значений.
        </Typography.Text>
    ));
});
```
