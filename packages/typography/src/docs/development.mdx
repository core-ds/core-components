## Подключение

Актуальный набор текстовых стилей доступен в нескольких вариантах:

1. Атомарные компоненты TitleDesktop, TitleMobile, TitleResponsive и Text
2. Компаунд-компоненты Typography.Title, Typography.TitleMobile, Typography.TitleResponsive и Typography.Text
3. Миксины в core-components/vars/typography.css

```jsx
// Атомарные импорты
import { TitleDesktop, TitleMobile, TitleResponsive, Text } from '@alfalab/core-components/typography';

// Компаунд импорт
import { Typography } from '@alfalab/core-components/typography';

@import '@alfalab/core-components/vars/typography.css';
```

## Системные шрифты

Если в макете используется шрифт San Francisco или Roboto, подключать его дополнительно не нужно.
Дизайнер ожидает, что на фронте в зависимости от платформы будет использоваться системный шрифт этой платформы.
Для использования необходимо назначить для body свойство  font-family со значением var(--font-family-system).

```jsx
@import '@alfalab/core-components/vars/typography.css';
```

## Styrene

Если в вашем продукте используется шрифт Styrene, его необходимо подключить.
Для всех Styrene-элементов должна быть включена настройка `font-feature-settings: 'ss01';`,
и рекомендуем не забыть про сглаживание `-webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale;`.

```css
@font-face {
    font-family: 'Styrene UI';
    src: url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_regular.woff2')
            format('woff2'), url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_regular.woff')
            format('woff');
    font-weight: 400;
    font-style: normal;
}
@font-face {
    font-family: 'Styrene UI';
    src: url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_medium.woff2') format('woff2'),
        url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_medium.woff') format('woff');
    font-weight: 500;
    font-style: normal;
}
@font-face {
    font-family: 'Styrene UI';
    src: url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_bold.woff2') format('woff2'),
        url('https://alfabank.servicecdn.ru/media/fonts/styrene-ui/styrene-ui_bold.woff') format('woff');
    font-weight: 700;
    font-style: normal;
}
```
